DoubleIntegratorWithAccel:

--------------------------------------------------------------------------
   Plant model for a double integrator with an acceleration state.

   State vector is [x;v;a]. Input vector is [0;0;da/dt]

   Since version 11.
--------------------------------------------------------------------------
   Form:
   [a, b] = DoubleIntegratorWithAccel( dT )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   dT       (1,1)  Time step

   -------
   Outputs
   -------
   a        (3,3)  State transition matrix
   b        (3,1)  Input matrix

--------------------------------------------------------------------------
