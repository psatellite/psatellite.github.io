MHTMatrixSortRows:

--------------------------------------------------------------------------
   Sort the matrix rows in ascending order. Move the first non-zero entry
   of the matrix to be in the first column.

--------------------------------------------------------------------------
   Form:
   [b2,ks] = MHTMatrixSortRows( b )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   b       (:,:)  Matrix representation of tracks

   -------
   Outputs
   -------
   b2      (:,:)   Matrix with rows sorted
   ks      (1,:)   Sort array

--------------------------------------------------------------------------
