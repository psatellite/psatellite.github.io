LocallyOptimalConeClock3D:

--------------------------------------------------------------------------
   Generates locally optimal cone and clock angles for 3D problems.

   Since version 7.
--------------------------------------------------------------------------
   Form:
   [a, cone, clock] = LocallyOptimalConeClock3D( x, type, d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x          (6,:)   x
   type       (1,:)   'cartesian' 'equinoctial'
   d          (1,1)   Data structure

   -------
   Outputs
   -------
   a          (3,:)   Acceleration vector 
   cone       (1,:)   Cone angle
   clock      (1,:)   Clock angle

--------------------------------------------------------------------------
