coranaFeval:

--------------------------------------------------------------------------
   Determines the value of the Corana function at point sol.
   This function is used in gademo2.

   Since version 7.
--------------------------------------------------------------------------
   Form:
   [val] = coranaEval(sol)
--------------------------------------------------------------------------

   ------
   Inputs
   ------  
   sol              the location to evaluate the Corana function        
  
   -------
   Outputs
   -------
   val              the value of the Corana function at point sol

--------------------------------------------------------------------------
