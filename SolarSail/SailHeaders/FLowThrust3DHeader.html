FLowThrust3D:

--------------------------------------------------------------------------
   This function is 3D orbit problem in equinoctial coordinates. 
   It is designed to be called by ode113. 

   Since version 7.
--------------------------------------------------------------------------
   Form:
   varargout = FLowThrust3D( t, x, flag, d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   t          (1,1)   Time
   x          (13,1)  States [p;f;g;h;k;L;m;lambda]
   flag       (1,1)   Flag from ode113
   d          (1,1)   Structure to pass to the integration RHS

   -------
   Outputs
   -------
   xDot       (13,1)  State derivatives d[r;u;v;lambdaR;lambdaU;lambdaV;v/r]/dt

   or

   vnew       (1,1)  Value of constraint
   isterminal (1,1)  1 if a terminal constraint  
   direction  (1,1)  Direction  

--------------------------------------------------------------------------

Children:

Sail: Optimization/ControlAngles3D
Sail: OrbitDynamics/RHSEquinoctial
