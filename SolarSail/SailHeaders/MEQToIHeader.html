MEQToI:

--------------------------------------------------------------------------
   Compute inclination from modified equinoctial coordinates.

   p = a(1 + e^2)
   f = ecos(w + W);
   g = esin(w + W);
   h = tan(i/2)cosW
   k = tan(i/2)sinW
   L = W + w + nu

   W is right ascension of the ascending node
   w is argument of perigee

   Since version 7.
--------------------------------------------------------------------------
   Form:
   i = MEQToI( x )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x          (6,1) Modified equinoctial elements

   -------
   Outputs
   -------
   i          (1,1) Inciination (rad)

--------------------------------------------------------------------------
