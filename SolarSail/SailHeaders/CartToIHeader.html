CartToI:

--------------------------------------------------------------------------
   Computes inclination from cartesian elements

   Since version 7.
--------------------------------------------------------------------------
   Form:
   i = CartToI( x, mu )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x               (6,1) [r;v]
   mu              (1,1) Gravitational constant

   -------
   Outputs
   -------
   i               (1,1) Inclination

--------------------------------------------------------------------------
