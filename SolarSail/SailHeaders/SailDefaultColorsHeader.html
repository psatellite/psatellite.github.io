SailDefaultColors:

--------------------------------------------------------------------------
   Define default colors for all sail plots.
   The options are: sun, cone, clock, clock line, force, body frame,
   and ref frame.

   Since version 7.
--------------------------------------------------------------------------
   Form:
   color = SailDefaultColors( action )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   action     (:)           Name

   -------
   Outputs
   -------
   color  (1,1) or (1,3)    Character or (R,G,B) triple 

--------------------------------------------------------------------------
