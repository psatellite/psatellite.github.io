SunGuidance:

--------------------------------------------------------------------------
   Guidance laws for common sun-centered profiles.
   The available methods are:

   'semi-major axis'
   'eccentricity'
   'inclination'
   'ascending node'
   'sun-pointing', 'sun'
   'edge-on', 'off'

   Since version 9.
--------------------------------------------------------------------------
   Form:
   [q,cone,clock,u] = SunGuidance( t, x, d, env )
--------------------------------------------------------------------------

   ------
   Inputs
   ------  
   t         (1,1)      Time
   x         (6,n)      Heliocentric state
   d          (:)       Data structure
                         .method   (:)   Method string
                         .dir     (1,1)  Direction flag
                         .mu      (1,1)  Gravitational parameter

   -------
   Outputs
   -------
   q        (4,n)       Inertial to body quaterion
   cone     (1,n)       Cone angle (rad)
   clock    (1,n)       Clock angle (rad)
   u        (3,n)       Sail normal vector in inertial frame

--------------------------------------------------------------------------
   See also LocallyOptimalTraj
--------------------------------------------------------------------------
