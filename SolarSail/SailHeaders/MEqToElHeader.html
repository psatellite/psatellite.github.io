MEqToEl:

--------------------------------------------------------------------------
   Transforms modified equinoctial elements to Keplerian elements.  
   The modified equinoctial elements are

   p = a(1 + e^2)
   f = ecos(w + W);
   g = esin(w + W);
   h = tan(i/2)cosW
   k = tan(i/2)sinW
   L = W + w + nu

   W is right ascension of the ascending node
   w is argument of perigee

   Since version 7.
--------------------------------------------------------------------------
   Form:
   el = MEqToEl( x, mu )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x              (6,:)  Modified equinoctial elements [p;f;g;h;k;L]
   mu             (1,1)  Gravitational parameter

   -------
   Outputs
   -------
   el             (6,:)  Keplerian elements [a;i;W;w;e;M]

--------------------------------------------------------------------------
