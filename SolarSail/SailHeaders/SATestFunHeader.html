SATestFun:

--------------------------------------------------------------------------
   Minimize the function f = 3x^3 + 5x^2 -2x + 3
   Given the constraint that -2 < x < 2

   Since version 7.
--------------------------------------------------------------------------
   Form:
   f = SATestFun( x, d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x              (1,1)  Value of x
   d              (1,1)  Structure to pass to fun (not used)

   -------
   Outputs
   -------
   f              (1,1)  Cost

--------------------------------------------------------------------------
