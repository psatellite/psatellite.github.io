FCartOrbD:

--------------------------------------------------------------------------
   Right hand side for cartesian orbital equations. Accepts a data structure
   input. See also FOrbCart.

   Since version 7.
--------------------------------------------------------------------------
   Form:
   xDot = FCartOrbD( x, t, d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x              (6,1)  [r;v] (km and km/s)
   d              (1,1)  Velocity vector (km/s)
                         .mu  (1,1) Gravitational parameter (km^3/s^2) 
                         .a   (3,1) Acceleration vector (km/s^2)

   -------
   Outputs
   -------
   xDot           (6,1)  d[r;v]/dt (km/s and km/s^2)

--------------------------------------------------------------------------
