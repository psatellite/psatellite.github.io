EnvironmentStruct:

--------------------------------------------------------------------------
   Return a default data structure for SailEnvironment.
   Will add the fields to an existing structure d0 if one is input.

   Since version 7.
--------------------------------------------------------------------------
   Form:
   d = EnvironmentStruct( d0 )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   d0            (:)    Data structure, optional

   -------
   Outputs
   -------
   d             (:)    Data structure with environment fields

--------------------------------------------------------------------------

Common: General/IsVersionAfter
