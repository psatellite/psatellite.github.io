SARosenbrock:

--------------------------------------------------------------------------
   The function f = 100*(x(1)^2-x(2))^2 + (1-x(1))^2
   Given the constraint that x(1) and x(2) be between
   -2.048 and 2.048.

   Since version 7.
--------------------------------------------------------------------------
   Form:
   f = SARosenbrock( x, d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x              (2,1)  Value of x
   d              (1,1)  Structure to pass to fun (not used)

   -------
   Outputs
   -------
   f              (1,1)  Cost

--------------------------------------------------------------------------
