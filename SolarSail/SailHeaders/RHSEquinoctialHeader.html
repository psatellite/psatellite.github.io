RHSEquinoctial:

--------------------------------------------------------------------------
   Right-hand-side for equinoctial elements.

   Since version 7.
--------------------------------------------------------------------------
   Form:
   xDot = RHSEquinoctial( t, x, d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   t          (1,1)  TIme
   x          (6,1)  States [p;f;g;h;k;L;lambda]
   d          (1,1)  Structure to pass to the integration RHS

   -------
   Outputs
   -------
   xDot       (6,1)   State derivatives d[p;f;g;h;k;L;lambda]/dt
   a          (6,6)   a matrix

--------------------------------------------------------------------------
