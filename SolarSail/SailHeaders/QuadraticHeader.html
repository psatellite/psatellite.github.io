Quadratic:

--------------------------------------------------------------------------
   The function f = x'*x

   Since version 7.
--------------------------------------------------------------------------
   Form:
   f = Quadratic( x, d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x              (:,1)  Value of x
   d              (1,1)  Data structure (unused)

   -------
   Outputs
   -------
   f              (1,1)  Cost

--------------------------------------------------------------------------
