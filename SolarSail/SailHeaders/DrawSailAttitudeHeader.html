DrawSailAttitude:

--------------------------------------------------------------------------
   View a rotated CAD model and the sun vector.
   If uSun has an extra column it is drawn as a planet vector.
   The sun and planet positions or sizes are not drawn to scale.

   Since version 7.
--------------------------------------------------------------------------
   Form:
   tag = DrawSailAttitude( g, q, uSun )
         DrawSailAttitude( g, q, uSun, tag )
--------------------------------------------------------------------------

   ------
   Inputs
   ------  
   g        (:)     CAD model data structure
   q        (4,1)   Inertial to body quaternion
   uSun     (3,n)   Sun/planet vector(s) in inertial frame
   tag      (:)     Window tag if updating an existing window

   -------
   Outputs
   -------
   tag      (:)     Window tag if initializing a window

--------------------------------------------------------------------------
