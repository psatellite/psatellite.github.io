RHSHelio2DOrbit:

--------------------------------------------------------------------------
   Planar heliocentric orbit problem with specular solar disturbance force. 
   Includes only radial position, radial velocity and tangential velocity.
   Has a built-in specular disturbance model.
   See also HeliopauseSimulation.

   Since version 7.
--------------------------------------------------------------------------
   Form:
   varargout = RHSHelio2DOrbit( t, x, flag, d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   t          (1,1)  Time
   x          (4,1)  States [r;u;v;angle]
   flag       (1,1)  Flag from ode113
   d          (1,1)  Structure to pass to the integration RHS
                     .distModel    (:)   Data structure
                     .sailAngleFun (:)   Angle function name
                     .forceModel   (:)   Either 'specular' or 'full' 
                     .laserOn      (1,1) 1 if laser is on
                     .laser
                                        .power    Power (W)
                                        .lambda   Wavelength (m)
                                        .aperture Optical aperture (m)

   -------
   Outputs
   -------
   xDot       (4,1)  State derivatives d[r;u;v;angle]/dt

--------------------------------------------------------------------------
