NewtonDemoFun:

--------------------------------------------------------------------------
   Returns the values of the equations

   0 = x(1) + x(2)^2 - 2
   0 = x(1)^2 - 1

   This is the built in demo for NewtonSolver.

   Since version 7.
--------------------------------------------------------------------------
   Form:
   y = NewtonDemoFun( x, d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x   (:,1) Initial guess
   d   (1,1) Data structure (not used)

   -------
   Outputs
   -------
   f   (:,1) Values

--------------------------------------------------------------------------
