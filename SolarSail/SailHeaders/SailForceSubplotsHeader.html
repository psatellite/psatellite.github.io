SailForceSubplots:

--------------------------------------------------------------------------
   View a single force vector in two planes using four subplots.
   Enter q to view clock angles. 
   See also QSail.

   Since version 7.
--------------------------------------------------------------------------
   Form:
   SailForceSubplots( f, n, s, q )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   f             (3,1)  Force vector
   n             (3,1)  Sail front normal vector
   s             (3,1)  Unit vector to sun
   q             (4,1)  Reference quaternion

   -------
   Outputs
   -------
   none

--------------------------------------------------------------------------
