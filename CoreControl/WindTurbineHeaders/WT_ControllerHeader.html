WT_Controller:

--------------------------------------------------------------------------
   Default Wind Turbine Controller. For use in WTSim.m.
--------------------------------------------------------------------------
   Forms:
     
     ioData = WT_Controller;                 % Obtain i/o data 
     output = WT_Controller( input );        % Update
     output = WT_Controller( input, time );  % Update with current sim time
     output = WT_Controller( input, -1 );    % initialize
   
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   input       (.)  Input data structure
   time        (1)  Simulation time

   -------
   Outputs
   -------
   output      (.)  Output data structure

--------------------------------------------------------------------------

Children:

WindTurbine: Utility/TorqueHAWT
