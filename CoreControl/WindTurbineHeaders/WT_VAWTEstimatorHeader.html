WT_VAWTEstimator:

--------------------------------------------------------------------------
   Default Wind Turbine Estimator. For use in WTSim.m.
--------------------------------------------------------------------------
   Forms:
     
     ioData = WT_Estimator;                 % Obtain i/o data 
     output = WT_Estimator( input );        % Update
     output = WT_Estimator( input, time );  % Update with current sim time
     output = WT_Estimator( input, -1 );    % initialize
   
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   input       (.)  Input data structure
   time        (1)  Simulation time

   -------
   Outputs
   -------
   output      (.)  Output data structure
