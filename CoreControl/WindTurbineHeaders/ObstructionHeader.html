Obstruction:

--------------------------------------------------------------------------
   Obstruction turbulent flow height.

   Type Obstruction for a demo.
--------------------------------------------------------------------------
   Form:
   z = Obstruction( zO, xO, x 
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   zO         (1,1)  Obstruction height
   xO         (1,1)  Obstruction width
   x          (1,:)  Position from obstruction can be negative

   -------
   Outputs
   -------
   z          (1,:)  Turbulent flow height
