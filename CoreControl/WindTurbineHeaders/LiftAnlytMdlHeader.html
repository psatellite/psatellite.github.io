LiftAnlytMdl:

--------------------------------------------------------------------------
   Computes lift using an analytical model

   For a demo, type LiftAnlytMdl.

--------------------------------------------------------------------------
   Form:
   cL = LiftAnlytMdl( alpha )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   alpha              (1,:) Angle of attack (rad)

   -------
   Outputs
   -------
   cL                 (1,:) Lift coefficient

--------------------------------------------------------------------------

Common: Graphics/Plot2D
