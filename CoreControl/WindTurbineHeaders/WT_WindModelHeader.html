WT_WindModel:

--------------------------------------------------------------------------
   Default Wind Turbine Wind Model. For use in WTSim.m.
--------------------------------------------------------------------------
   Forms:
     
     ioData = WT_WindModel;                 % Obtain i/o data 
     output = WT_WindModel( input );        % Update
     output = WT_WindModel( input, time );  % Update with current sim time
     output = WT_WindModel( input, -1 );    % initialize
   
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   input       (.)  Input data structure
   time        (1)  Simulation time

   -------
   Outputs
   -------
   output      (.)  Output data structure

--------------------------------------------------------------------------
