WT_WindSensor:

--------------------------------------------------------------------------
   Default Wind Turbine Wind Sensor Model. For use in WTSim.m.
--------------------------------------------------------------------------
   Forms:
     
     ioData = WT_WindSensor;                 % Obtain i/o data 
     output = WT_WindSensor( input );        % Update
     output = WT_WindSensor( input, time );  % Update with current sim time
     output = WT_WindSensor( input, -1 );    % initialize
   
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   input       (.)  Input data structure
   time        (1)  Simulation time

   -------
   Outputs
   -------
   output      (.)  Output data structure

--------------------------------------------------------------------------
