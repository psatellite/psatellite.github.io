HAWTRHS:

--------------------------------------------------------------------------
   Horizontal Axis Wind Turbine Dynamics.
   This model includes an electrical machine in direct-quadrature form.
--------------------------------------------------------------------------
   Form:
   xDot = HAWTRHS( x, t, d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x          (4,1)  State [angle;rate;iD;iQ]
   t          (1,1)  Time (not used)
   d          (1,1)  Data structure
                     .rho      (1,1) Atmospheric density (kg/m^3
                     .rTurbine (1,1) Blade radius (m)
                     .wind     (1,1) Wind speed (m/s)
                     .beta     (1,1) Pitch angle (rad)
                     .uD       (1,1) Direct voltage
                     .uQ       (1,1) Quadrature voltage
                     .inr      (1,1) Inertia

   -------
   Outputs
   -------
   xDot       (4,1)  State derivative [angle;rate;iD;iQ]

--------------------------------------------------------------------------

Children:

Electrical: Motor/PMBMachineRHS
WindTurbine: Utility/TorqueHAWT
