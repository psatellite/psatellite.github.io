RHSBatchR:

--------------------------------------------------------------------------
   Model of a batch reactor
--------------------------------------------------------------------------
   Form:
   xDot = RHSBatchR( x, t, d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x           (2,1) State [CA;T]
   t           (1,1) Time
   d           (1,1) data structure
                     .dH   (1,1) Delta H/rho*C liter - deg-C/gmole
                     .uA   (1,1) UA/VrhoC 1/s
                     .tC   (1,1) mol/L
                     .k0   (1,1) liter/gmole-sec
                     .eAR  (1,1) Ea/R deg-C
   -------
   Outputs
   -------
   xDot        (2,1) State derivative

--------------------------------------------------------------------------
   Reference: Henson, M. A. and D. E. Seborg. (1997.) Nonlinear 
              Process Control, Prentice-Hall. pp. 312-313.
--------------------------------------------------------------------------
