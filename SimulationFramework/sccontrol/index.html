Spacecraft Control Library

This application programming interface (API) document contains documentation for functions in SCControl framework.
There are modules for coordinate transformation, modeling the space environment, computing ephemeris, and performing orbit mechanics including planetary perturbations. This library utilizes the broad range of matrix manipulation functions provided in MatrixLib. No manufacturers' data is supplied with the models.

The formation flying module, if included, is a comprehensive set of functions for the guidance and control of formation flying spacecraft.
