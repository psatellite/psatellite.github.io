Detailed Description

Missile aerodynamic model. The organization of the data file is m Number of models [ x;y;z ] Rotation axis of model [ x;y;z ] Location of model with respect to the reference If the rotation axis is [0;0;0] then the model is assumed not to rotate relative to the core body.
