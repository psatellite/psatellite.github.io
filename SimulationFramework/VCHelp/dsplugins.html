Data Source Plugins

One of the four types of plugins available to VisualCommander, Data Source Plugins provide new ways of getting data into VisualCommander. The possibilities are endless: a Data Source Plugin could run simulations, control test hardware, retrieve data from the internet or command remote systems. The end user can create whatever plugin is necessary to fulfill their needs and be assured that VisualCommander's visualization tools can be leveraged to help manage it.
