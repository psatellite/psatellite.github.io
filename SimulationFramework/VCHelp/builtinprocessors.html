Built-in Processor Plugins

Processor plugins allows you to process raw data in a variety of ways. VisualCommander comes with a set of general-purpose but very useful processor plugins. This page explains how to use the builtin display plugins.

Summary

To see the list of built-in processor plugins from Visual Commander, go to Window->Tools, or hit CMD+R. The following window of "Tools" appears.

The processor plugins or "Tools" are summarized below by category.

Basic

  • Constant - Create a constant.
  • Statistics (Basic) - Create a sequence of commands.
  • Statistics (Matrix) - Choose from a fixed set of values.
  • Time - Send a configured value when the button is pressed.

Matrix

  • Composition - Create a matrix.
  • Cross Product - Cross product of two 3 element column matrices.
  • Dimensions -Dimensions of the input matrix.
  • Euclidean Distance - Distance between two points.
  • Submatrix - Get a submatrix from a matrix.
  • Transpose - Transposes the input matrix.
  • Unit - Unitizes the input matrix.
  • Vector - Magnitude of an input matrix.

Quaternion

  • Detransform - Transformation in the direction of the transpose of a quaternion.
  • Multiply - Multiply two quaternions q(a to b) x q(b to c) = q(a to c).
  • Quaternion transpose - Transpose of a quaternion.
  • Transform - Transform a vector in the diretion of the quaternion.

Script

  • Expression Evaluator Evaluate a script containing a series of operations.

Using Processor Plugins

You drag processors onto the data flow page and make connections.
