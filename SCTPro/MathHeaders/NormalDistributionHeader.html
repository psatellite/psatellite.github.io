NormalDistribution:

Path: Math/Probability

% Matrix normal or Gaussian distribution.

   Type NormalDistribution for a demo.

   Since version 11.
--------------------------------------------------------------------------
   Form:
   n = NormalDistribution( x, m, sigma )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x       (n,1)  Value
   mu      (n,1)  Mean (can be (1,1)
   sigma	(n,n)  Standard deviation

   -------
   Outputs
   -------
   p       (n,1) Normal distribution
