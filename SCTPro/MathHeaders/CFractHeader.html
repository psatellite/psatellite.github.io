CFract:

Path: Math/Analysis

% Computes the value of a continued fraction of the form
 a0
 ----------------------
 b0 - a1
         -----------------
         b1 - a2
                ------------
                b2 - a3
                       -------
                       b3

 The maximum accuracy depends on the size of a and b. tol is only
 used for early termination of the series.

--------------------------------------------------------------------------
   Form:
   x = CFract( a, b, tol )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   a                     a array
   b                     b array
   tol                   Error tolerance

   -------
   Outputs
   -------
   x                     value of fraction

--------------------------------------------------------------------------
   References: Battin, R. H., An Introduction to the Mathematics and
               Methods of Astrodynamics, AIAA, 1987, pp. 67-68.
--------------------------------------------------------------------------
