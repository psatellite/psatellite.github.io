BilinearInterpolation:

Path: Math/Geometry

% Apply bilinear interpolation on 4 data points.

   Type BilinearInterpolation for a demo.

   Since version 2014.1
--------------------------------------------------------------------------
   Form:
   zi = BilinearInterpolation( x, y, z, xi, yi );
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x   (1,2) Values
   y   (1,2) Values
   z   (2,2) Values
   xi  (1,:) Inputs
   yi  (1,:) Inputs

   -------
   Outputs
   -------
   zi  (:,:) Interpolated values

--------------------------------------------------------------------------
