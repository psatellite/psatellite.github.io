SphJacobian:

Path: Math/Analysis

% Spherical coordinates Jacobian
 This function is the application to the chain rule to a function U in
 spherical coordinates U( r, phi, theta ) such that

 [dU/dx;dU/dy;dU/dZ] = c*[dU/dr;dU/dphi;dU/dtheta]

--------------------------------------------------------------------------
   Form:
   c = SphJacobian( x )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   r            (3,1)   Vector
                        
   -------
   Outputs
   -------
   c            (3,3)   Jacobian
                        
--------------------------------------------------------------------------
