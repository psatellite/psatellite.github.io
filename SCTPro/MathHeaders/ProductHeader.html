% Computes product of the terms in a vector
   If a matrix it computes products of the terms in each row and
   returns a column vector.

   Since version 1.
--------------------------------------------------------------------------
   Form:
   p = Product( v )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   v                  Vector

   -------
   Outputs
   -------
   p                  Product
