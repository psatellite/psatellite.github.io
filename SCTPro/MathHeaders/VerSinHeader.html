VerSin:

Path: Math/Trigonometry

% Produces a smoothly varying function from 0 to xMax. 
   The minimum value of y is zero and the maximum value is 1.

   Since version 2.
--------------------------------------------------------------------------
   Form:
   y = VerSin( x, xMax )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x            (:)       The vector
   xMax                   Maximum value

   -------
   Outputs
   -------
   y            (:)       Output

--------------------------------------------------------------------------
