% Compute the probability density function for a Gaussian distribution.
--------------------------------------------------------------------------
   Form:
   f = GaussianPDF( x, mu, sigma )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x               (1,:)    Value
   mu              (1,1)    Mean
   sigma           (1,1)    Standard deviation

   -------
   Outputs
   -------
   f               (1,:)    Value
