TrigReduction:

Path: Math/Trigonometry

% Solve a*sin(beta) + b*cos(beta) = c.
   For a real answer c^2 < a^2 + b^2.

   Type TrigReduction for a demo.

   Since version 10.
--------------------------------------------------------------------------
   Form:
   beta = TrigReduction( a, b, c )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   a                  Coefficient
   b                  Coefficient
   c                  Right hand side

   -------
   Outputs
   -------
   beta               Angle solution (rad)

--------------------------------------------------------------------------
