GJInv:

Path: Math/Linear

% Matrix inverse using Gauss-Jordan elimination with partial pivoting.

   Since version 2.
--------------------------------------------------------------------------
   Form:
   b = GJInv( a )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   a			  (n,n)  Matrix

   -------
   Outputs
   -------
   b			  (n,n)  Inverse

--------------------------------------------------------------------------
	References:	Strang, G.S., "Linear Algebra and its Applications Third Edition"
               Harcourt, Brace and Jovanovich, 1988, pp. 42-58.
