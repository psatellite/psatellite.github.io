BlockDiagonal:

Path: Math/Linear

% Generate a block diagonal matrix

 Puts m0 along the diagonal.

--------------------------------------------------------------------------
   Form:
   m = BlockDiagonal( m0, n )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   m0	(m,m)     Matrix
   n 	(1,1)    Number of blocks

   -------
   Outputs
   ------- 
   m	(n*m,n*m)   Block diagonal matrix

--------------------------------------------------------------------------
