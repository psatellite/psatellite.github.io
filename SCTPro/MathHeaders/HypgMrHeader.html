HypgMr:

Path: Math/Analysis

% Computes the value of the confluent hypergeometric ratio

 M(beta+1,gamma+1,x)
 -------------------
 M(beta,gamma,x)

--------------------------------------------------------------------------
   Form:
   [F, n] = HypgMr( beta, gamma, x )
   [F, n] = HypgMr( beta, gamma, x,tol )
   [F, n] = HypgMr( beta, gamma, x, tol, nMax )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   beta                  Argument
   gamma                 Argument
   x                     Argument
   tol                   Tolerance
   nMax                  Maximum number of iterations

   -------
   Outputs
   -------
   F                     Value of fraction
   n                     Number of iterations

--------------------------------------------------------------------------
   References: Battin, R. H., An Introduction to the Mathematics and
               Methods of Astrodynamics, AIAA, 1987, pp. 34-68.
--------------------------------------------------------------------------
