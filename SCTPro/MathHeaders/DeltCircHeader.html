DeltCirc:

Path: Math/Trigonometry

% Find the angle between x and y. 

   This function is for subtracting numbers on a circle.

   Handle all input angles so that the solution is never greater
   than 2*pi.

   For example, DeltCirc( 2*pi, pi/6 )

--------------------------------------------------------------------------
   Form:
   z = DeltCirc( x, y )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x                  Angle
   y                  Angle

   -------
   Outputs
   -------
   z                  Delta angle

--------------------------------------------------------------------------
