Interp1D:

Path: Math/Analysis

% Performs a linear interpolation:
 y = (y2-y1)*(x-x1)/(x2-x1) + y1

--------------------------------------------------------------------------
   Form:
   y = Interp1D( y1, y2, x1, x2, x )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   y1                 First value
   y2                 Second value
   x1                 First value
   x2                 Second value
   x                  Input

   -------
   Outputs
   -------
   y                  Interpolated value

--------------------------------------------------------------------------
