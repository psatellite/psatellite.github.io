MeasurementsSunMagUKF:

Path: SpacecraftEstimation/SensorMeas

% Outputs the set of acceptable attitude measurements from vectors.
 Computes measurements given a pixel map from the unit vector
 measurements. This function is for use with the UKF as it 
 does not return a linearized measurement matrix.
--------------------------------------------------------------------------
   Form:
   y = MeasurementsSunMagUKF( x, d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x   (6,1)	[dQ;b]
   d   (.) 	Data structure

   -------
   Outputs
   -------
   y	(:,1)    Measurements

--------------------------------------------------------------------------
