EarthBc:

Path: SpacecraftEstimation/StellarEphem

% Compute the barycentric position and velocity of the earth.

   Type EarthBc for a demo

   Since version 3.
--------------------------------------------------------------------------
   Form:
   [eB, eBDot] = EarthBc( jD )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   jD              (1,:)    Julian date
   

   -------
   Outputs
   -------
   eB              (3,:)    Barycentric position (aU)
   eBDot           (3,:)    Barycentric rate (aU/y)

--------------------------------------------------------------------------
