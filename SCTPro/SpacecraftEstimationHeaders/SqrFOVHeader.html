Path: SpacecraftEstimation/StellarEphem

% Determines if a unit vector falls within a square FOV
 Determines if a unit vector falls within a square x by y field of view. The
 boresight is along +z.
------------------------------------------------------------------------
   Form:
   k = SqrFOV( u, x, y )
------------------------------------------------------------------------

   ------
   Inputs
   ------
   u            (3,:)     Unit vectors
   x            (1,1)     x width (rad)
   y            (1,1)     y width (rad)

   -------
   Outputs
   -------
   k            (1,:)     Columns that are within the field-of-view

------------------------------------------------------------------------
