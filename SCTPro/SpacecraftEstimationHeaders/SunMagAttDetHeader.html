SunMagAttDet:

Path: SpacecraftEstimation/AttitudeEstimation

% An attitude determination system using vector measurements. 
   The first column of catalog is the magnetometer data.
--------------------------------------------------------------------------
   Form:
   [x, p] = SunMagAttDet( x, p, r, qBToS, bMeas, sensorData, catalog, fScale )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x            (3,1)     Attitude [x;y;z]
   p            (3,3)     State covariance matrix
   r            (m)       Measurement noise
   qBToS        (4,1)     Quaternion from body to sensor axes
   bMeas        (3,1)     Measured magnetic field
   sensorData   (m,3)     .xy        (2,:) Focal plane data
                          .validMeas (:)   1 if measurement is valid
   catalog      (3,p)     Catalog of unit vectors
   fScale                 Sensor scale factor

   -------
   Outputs
   -------
   x            (3,1)     Attitude [x;y;z]
   p            (3,3)     State covariance matrix

--------------------------------------------------------------------------
