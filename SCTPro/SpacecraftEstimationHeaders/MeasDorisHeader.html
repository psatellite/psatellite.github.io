MeasDoris:

Path: SpacecraftEstimation/OrbitEstimation

% Range rate to an Earth target.
 Includes Earth rotational rate. This is a simple model of a Doris
 sensor. It measures Doppler shift, producing range rate. It is 
 designed for use by UKFUpdate.

--------------------------------------------------------------------------
   Form:
   [y, rT] = MeasDoris( x, d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x   (3,1)	Position of spacecraft (km)
   d   (.)   Data structure
             .jD0          (1,1) Julian date
             .omegaSkew	  (3,3) Skew symmetric matrix for earth rate
             .t           	(1,1) Time (s)
             .rEF          (3,1) Earth fixed target location
             .noise1Sigma  (1,1) range rate 1 sigma (km,km/s)

   -------
   Outputs
   -------
   y 	(1,1) range rate
   rT 	(3,1) Target on Earth in ECI frame (km)

--------------------------------------------------------------------------
