SelectMeasurementSet:

Path: SpacecraftEstimation/StellarAttDet

% Select a subset of measurements.
--------------------------------------------------------------------------
   Form:
   [measPairsSubset, hitPairsSubset, measMap] = SelectMeasurementSet( firstMeas,
                                                   nMeas, measPairs, hitPairs )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   kSet            (1,:)     Set measurement to be used
   nMeas           (1,1)     Number of measurements to use
   measPairs       (2,n)     Column is a measurement. Each row gives the meas id
   hitPairs        {n}       Dot product hits. Each two numbers gives a star pair
                             [star1_meas1 star2_meas1 star1_meas2 star2_meas2...]

   -------
   Outputs
   -------
   measPairsSubset (2,n)     Column is a measurement. Each row gives the meas id
   hitPairsSubSet  {n}       Dot product hits. Each two numbers gives a star pair
                             [star1_meas1 star2_meas1 star1_meas2 star2_meas2...]
   measMap         (1,:)     Maps measurement pair subset to input measurements

--------------------------------------------------------------------------
