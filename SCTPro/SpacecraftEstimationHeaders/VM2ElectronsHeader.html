VM2Electrons:

Path: SpacecraftEstimation/StellarCatalog

% Converts apparent visual magnitude to electrons per second per mm^2.

   Since version 8.
------------------------------------------------------------------------
   Form:
        VM2Electrons;
        VM2Electrons( v );
        VM2Electrons( v, c );
   e =  VM2Electrons( v )
   e =  VM2Electrons( v, c )
------------------------------------------------------------------------

   ------
   Inputs
   ------
   v                 (:)    Visual magnitude
   c                 (1,1)  Amps/W

   -------
   Outputs
   -------
   e                 (:)    Electrons (e-/s/mm^2)

------------------------------------------------------------------------
