--------------------------------------------------------------------------

   xDot = FOrbitKFPointMass( x, t, d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x            (6,1)     The state vector [r;v]
   t            (1,1)     Time (sec)

   Outputs
   -------
   xDot         (6,1)     The derivative of the state vector
