% Quaternion state transition matrix
 Generates the state transition matrix to propagate a quaternion one step
 assuming that the angular velocity is constant.
------------------------------------------------------------------------
   Form:
   phi = STM( w, dT )
------------------------------------------------------------------------

   ------
   Inputs
   ------
   w            (3,1)     Angular rate
   dT           (1,1)     Time change

   -------
   Outputs
   -------
   phi          (4,4)     State transition matrix
