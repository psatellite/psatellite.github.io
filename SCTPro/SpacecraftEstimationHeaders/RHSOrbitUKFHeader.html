% Right-hand-side for a cartesian orbit.
   t is not used. This is compatible with RK4 and RK45.
--------------------------------------------------------------------------
   Form:
   f = RHSUKF( x, t, d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x         (6,1)  State [r;v]
   t         (1,1)  Time (not used)
   d         (1,1)  Data structure
                    .a   (3,1) Acceleration
                    .mu  (1,1) Gravitational parameter

   -------
   Outputs
   -------
   f         (6,1)  State derivative

--------------------------------------------------------------------------
   References: None
--------------------------------------------------------------------------
