SunBc:

Path: SpacecraftEstimation/StellarEphem

% Compute the barycentric position of the sun. 
 This function currently returns [0;0;0] which is in error by about 0.002 Au.
------------------------------------------------------------------------
   Form:
   sB = SunBc( jD )
------------------------------------------------------------------------

   ------
   Inputs
   ------
   jD                (1,1)  Julian date
   

   -------
   Outputs
   -------
   sB                (3,1)  Barycentric position (aU)
