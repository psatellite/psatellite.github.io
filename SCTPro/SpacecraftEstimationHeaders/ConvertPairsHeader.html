ConvertPairs:

Path: SpacecraftEstimation/StellarAttDet

% Convert pairs from the old format to the new format
 Old format:

  hitPairs{k}{i} = [star1_pair1 star1_pair2 ... star1_pairN;...
                   star2_pair1 star2_pair2 ... star2_pairN

 To the new format:

  hitPairs = {[ star1_pair1 star2_pair1 star1_pair2 star2_pair2 ... ],... % meas pair 12
              [ star1_pair1 star2_pair1 star1_pair2 star2_pair2 ... ],... % meas pair 13
	           ...
              [ star1_pair1 star2_pair1 star1_pair2 star2_pair2 ... ]}    % meas pair nm

--------------------------------------------------------------------------
   Form:
   [hitPairs = ConvertPairs( hitPairsOld )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   hitPairsOld    {n,m}  Dot product hits. Each ell

   -------
   Outputs
   -------
   hitPairs       {1,1}  Dot product hits. Each two numbers gives a star pair
                         [star1_meas1 star2_meas1 star1_meas2 star2_meas2...]

--------------------------------------------------------------------------
