EllipseToS:

Path: Imaging/Optics

% Convert ellipse parameters to s.

   Type EllipseToS for a demo

--------------------------------------------------------------------------
   Form:
   s = EllipseToS( a, b, x0, y0 )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   a         (1,1) x-axis width
   b         (1,1) y-axis width
   x0        (1,1) x-axis origin
   y0        (1,1) y-axis origin

   -------
   Outputs
   -------
   s         (1,6) Coefficients of [x^2 y^2 xy x y 1]
