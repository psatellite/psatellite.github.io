CreateLensTable:

Path: Imaging/Utilities

% Creates a lens table from l. 

 If the fileName is not entered it will generate a table in the
 workspace. Otherwise, it will create a latex table.

 Type CreateLensTable for a demo.
--------------------------------------------------------------------------
   Form:

   CreateLensTable( x, fileName )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   l       	(:)	Lens data structure
   fileName	(:)	File name or file id

   -------
   Outputs
   -------
   None
