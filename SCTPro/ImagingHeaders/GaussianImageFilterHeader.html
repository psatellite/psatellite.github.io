GaussianImageFilter:

Path: Imaging/ImageProcessing

% Apply a Gaussian filter and derivative to an image.

--------------------------------------------------------------------------
   Form:
   iF = GaussianImageFilter( q, sigma, eps, y )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   q              (n,m)       Grayscale image
   sigma          (1,1)       Standard deviation
   eps            (1,1)       Size of filter box; default, 0.01
   y              (1,1)       If entered filter along y

   -------
   Outputs
   -------
   qF             (n,m)       Processed image

--------------------------------------------------------------------------
