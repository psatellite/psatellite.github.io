EffFocalLength:

Path: Imaging/Optics

% Computes the effective focal length using thin lens approximation.
   d(1) is the distance between lens 1 and lens 2.

   Type EffFocalLength for a demo.

--------------------------------------------------------------------------
   Form:
   fE = EffFocalLength( f, d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   f             (1,n)   Focal length
   d             (1,n-1) Distance between lenses

   -------
   Outputs
   -------
   fE            (1,1)   Effective focal length
