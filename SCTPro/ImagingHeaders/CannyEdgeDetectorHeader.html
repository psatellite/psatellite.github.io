CannyEdgeDetector:

Path: Imaging/ImageProcessing

% Applies a mask to do Canny edge detection.
 Uses a 5x5 mask.

 Type CannyEdgeDetector for a demo with the asteroid Bennu.
--------------------------------------------------------------------------
   Form:
    c = CannyEdgeDetector(a)
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   a              (n,m)        Grayscale image

   -------
   Outputs
   -------
   c              (n,m)        Edges

--------------------------------------------------------------------------
 See also ConvolveImage
