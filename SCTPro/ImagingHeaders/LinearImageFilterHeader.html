LinearImageFilter:

Path: Imaging/ImageProcessing

% Apply a linear noise filter to an image.
 Has a built-in demo using MoonAndStars.jpg.
--------------------------------------------------------------------------
   Form:
        LinearImageFilter;         % demo
   iF = LinearImageFilter( i, a )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   i              (:,:)       Grayscale image (double)
   a              (n,n)       Mask

   -------
   Outputs
   -------
   iF             (:,:)       Filtered image

   See also conv2.
--------------------------------------------------------------------------
