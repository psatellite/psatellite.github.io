RayTraceMirror:

Path: Imaging/Optics

% Ray trace for mirrors.
--------------------------------------------------------------------------
   Form:
   r = RayTraceMirror( r, l )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   r         (:)     Ray source definition
                     .p     (2,1) Origin
                     .m     (2,1) Ray unit vector
   l         (:)     Mirror data structure
                     .s     (1,6) [x^2 y^2 xy x y 1]
                     .sa    (1,1) Semi aperture (radius about x)
                     .r     (1,1) Radius of hole in mirror

   -------
   Outputs
   -------
   r         (:)     Ray source definition
                     .p     (2,:) Origins
                     .m     (2,:) Ray unit vector

--------------------------------------------------------------------------
