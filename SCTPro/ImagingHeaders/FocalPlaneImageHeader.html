FocalPlaneImage:

Path: Imaging/Optics

% Visualize a focal plane image using a mesh.
   Initialize creates a new figure, which is stored in a persistent
   variable. Update will then update the image in this figure.

   Type FocalPlaneImage for a demo with random pixel intensities. 

--------------------------------------------------------------------------
   Form:
   FocalPlaneImage( 'initialize', e )
   FocalPlaneImage( 'update', p, k )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   action      (1,:)   'initialize' 'update'
   action = 'initialize'
      e            (.)    Data structure
                           .m    (1,1) pixels
                           .n    (1,1) pixels
   action = 'update'
      p           (n,m)   Pixel intensities
      k           (1,1)   Frame index

   -------
   Outputs
   -------
   None

--------------------------------------------------------------------------

Children:

Common: Graphics/NewFig
