RHSGaussianPSF:

Path: Imaging/PSFUtils

% Point Spread Function for multiple Gaussian PSF

 $$ f = i \exp(-((x - \mu_x)^2 + (y - \mu_y)^2)/(2\sigma^2)) $$

--------------------------------------------------------------------------
   Form:
   f = RHSGaussianPSF(x,y,d)
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x   (:,:)       X coordinates
   y   (:,:)       Y coordinates
   d    (.)        Data structure
                   .d    (.)  Data structure for PSF
                                 .sigma (1,1) Standard deviation
                   .r   (2,:) Centers of PSF
                   .i   (1,:) Intensities of PSF
   -------
   Outputs
   -------
   f   (n,m)       Pixel output

--------------------------------------------------------------------------
