LumenToWatt:

Path: Imaging/Optics

% Outputs lumens/watt.

   If lambda is an array, l must be a scalar. If lambda is a scalar
   l can be an array.

   Type LumenToWatt to get the conversion curve.

--------------------------------------------------------------------------
   Form:
   w = LumenToWatt( lambda, l )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   lambda    (1,:) Wavelength (micro-m)
   l         (1,:) Lumens

   -------
   Outputs
   -------
   w         (1,:) Watts or lumens/watt

--------------------------------------------------------------------------
