CombineOBJFiles:

Path: AerospaceUtils/CAD

% Combines obj files and writes a new file.
 Creates both an obj file and mtl file. Don't enter the suffixes; enter xxx not
 xxx.obj. The vertices may be rotated and translated. After rotation delta is
 subtracted from the rotated vertex positions i.e. v = b*v - delta.

 If the fileNew does not include a path, it will be saves at the same location
 as the first file in the list.
--------------------------------------------------------------------------
   Form:
   CombineOBJFiles( file, fileNew, b, scale, delta, flip )
--------------------------------------------------------------------------

   ------
   Inputs
   ------

   file              {n}(1,:) Filenames
   fileNew           (1,:)    New model file
   b                 {:}(3,3) Transformation matrices
   scale             (1,n) Optional scale factor for the model
   delta             (3,n) Offset applied after scaling and rotation
   flip              (1,n) 1 if flip faces

   -------
   Outputs
   -------
   None

--------------------------------------------------------------------------
