PartialThickDisk:

Path: AerospaceUtils/ComponentModels

% CAD vertices and faces for a partial thick disk.

 Type PartialThickDisk  for a demo.

--------------------------------------------------------------------------
   Form:
   [v, f] = PartialThickDisk( rL, r0, l, n, thetaStart, thetaEnd )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   rL           (1,1) Top radius    (z = l)
   r0           (1,1) Bottom radius (z = 0)
   l            (1,1) Length
   n            (1,1) Number of divisions
   angleStart   (1,1) Start angle (default 0)
   angleEnd     (1,1) End angle 

   -------
   Outputs
   -------
   v           (:,3) Vertices
   f           (:,3) Faces

--------------------------------------------------------------------------
