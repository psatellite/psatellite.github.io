% Generate CAD vertices and faces for an ESPA bus

 Type ESPABus for a demo.
--------------------------------------------------------------------------
   Form:
   [v,f,d] = ESPABus(type,b)
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   type (1,:) 'grande' or 'other'
   b    (3,3) Transformation matrix 

   -------
   Outputs
   -------
   v           ( 8,3) Vertices
   f           (12,3) Faces
   d           (1,3)  [x y z] dimensions (original orientation)

--------------------------------------------------------------------------
 See also: ESPAArray
