% Vertices and faces for an ESPA Array

 Type ESPAArray for a demo. If there are not outputs it will draw the
 model.
--------------------------------------------------------------------------
   Form:
   [vA,fA,d] = ESPAArray(type,nX,nY)
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   type  (1,:) 'grande' or 'other'
   nX    (1,1) Number in x
   nY    (1,1) Number in y

   -------
   Outputs
   -------
   vA    (:,3) Vertices
   fA    (:,3) Faces (original orientation)
   d     (1,3) Dimensions

--------------------------------------------------------------------------
 See also: ESPABus
