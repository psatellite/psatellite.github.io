Path: AerospaceUtils/ComponentModels

% Get CAD vertices for an RWA.
 Type RWAPatch for a demo.
--------------------------------------------------------------------------
   Form:
   [v, f] = RWAPatch( r, u )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   r           (1,:) Type or struct:
                     ('r','hCenter','rTop','hTop', 'rBottom', 'hBottom', 'u')
   u           (3,1) Unit vector is r is a character.

   -------
   Outputs
   -------
   v           (:,3) Vertices
   f           (:,3) Faces

--------------------------------------------------------------------------
  See also: RWAModel
