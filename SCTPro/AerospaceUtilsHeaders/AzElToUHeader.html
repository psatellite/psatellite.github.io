AzElToU:

Path: AerospaceUtils/Coord

% Converts azimuth and elevation to a unit vector.

 Type AzEleToU for a demo.

--------------------------------------------------------------------------
   Form:
   u = AzElToU( az, el )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   az        (1,:)   Azimuth.    
   el        (1,:)   Elevation.    

   -------
   Outputs
   -------
   u         (3,:)   Unit vectors

--------------------------------------------------------------------------
