NoseCone:

Path: AerospaceUtils/ComponentModels

% Get CAD vertices and faces for a nose cone.

 Type NoseCone for a demo.
--------------------------------------------------------------------------
   Form:

   [v, f] = NoseCone( r, l, n, h )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   r            (1,:) R as a function of z
   l            (1,1) Length
   n            (1,1) Number of divisions
   h            (1,1) 1 if half cone

   -------
   Outputs
   -------
   v            (:,3) Vertices
   f            (:,3) Faces

--------------------------------------------------------------------------
