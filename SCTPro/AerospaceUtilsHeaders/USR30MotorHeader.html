USR30Motor:

Path: AerospaceUtils/ComponentModels

% Generate CAD vertices and faces for a motor model.

 Type USR30Motor for a demo.

 Since version 10.
--------------------------------------------------------------------------
   Form:
   [v, f, d] = USR30Motor( demo )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   demo        (1,1) If entered run the demo

   -------
   Outputs
   -------
   v           (:,3) Vertices
   f           (:,3) Faces
   d           (1,1) Data
                     .r (1,3) Radii
                     .z (1,3) Z locations of base

--------------------------------------------------------------------------
