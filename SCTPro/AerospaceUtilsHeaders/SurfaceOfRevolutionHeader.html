SurfaceOfRevolution:

Path: AerospaceUtils/ComponentModels

% Generate CAD vertices and faces for a surface of revolution about z.
 Type SurfaceOfRevolution for a demo.
--------------------------------------------------------------------------
   Form:
   [v, f] = SurfaceOfRevolution( r, z, n )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   r	(1,:)   Radius at each z
   z	(1,:)   z length
   n	(1,1)   Number of segments per revolution (Default 20)
   

   -------
   Outputs
   -------
   v	(:,3)   Vertices
   f	(:,3)   Faces

--------------------------------------------------------------------------
