QSLERP:

Path: AerospaceUtils/Coord

% Interpolate a quaternion using the SLERP algorithm.
 Interpolates between qFrom and qTo using the scaled time t
--------------------------------------------------------------------------
   Form:
   q = QSLERP( qFrom, qTo, t )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   qFrom (4,1) Starting quaterion
   qTo   (4,1) Final quaternion
   t     (1,1) 0 to 1

   -------
   Outputs
   -------
   q     (4,1) Quaternion   

--------------------------------------------------------------------------
