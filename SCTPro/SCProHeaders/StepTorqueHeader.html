StepTorque:

Path: SCPro/DynamicalModels

% Produces a step torque. 

 This is the standard format for an disturbance torque functions.
--------------------------------------------------------------------------
   Form:
   t = StepTorque( ~, ~, d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x   (:,1)	State (unused)
   t   (1,1)	Time (unused)
   d 	(.)   Data structure
             .torqueDisturbance

   -------
   Outputs
   -------
   t   (3,1)       Torque

--------------------------------------------------------------------------
