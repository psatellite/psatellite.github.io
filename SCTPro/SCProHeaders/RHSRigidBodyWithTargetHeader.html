RHSRigidBodyWithTarget:

Path: SCPro/DynamicalModels

% Right-hand-side of spacecraft dynamical equations with a second spacecraft.
   This is a rigid body with an external force and torque input.
   The position is in km. The gravity model is for a point mass.
   The prime spacecraft has attitude dynamics, the target has only
   point mass dynamics (position and velocity.)
   The force and torque are constant across the integration time step.

--------------------------------------------------------------------------
   Form:
   d = RHSRigidBodyWithTarget;
   xDot = RHSRigidBodyWithTarget( x, t, d )
--------------------------------------------------------------------------

   -------
   Inputs
   -------
   x       (:,1)	State vector [r;v;q;w;rT;vT] (km,km/s,,rad/s)
   t       (1,1)	Time (unused) (s)
   d       (1,1)	Data structure for the simulation
                   .torque   (3,1) External torque (Nm)
                   .force    (3,1) External force (N)
                   .inertia  (3,3) Vehicle inertia (kg-m^2)
                   .mass     (1,1) Vehicle mass (kg)
                   .mu       (1,1) Gravitational parameter (km^3/s^2)

   -------
   Outputs
   -------
   xDot	(:,1)     d[r;v;q;w;rT;vT]/dt

--------------------------------------------------------------------------
