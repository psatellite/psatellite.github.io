AsteroidProperties:

Path: SCPro/ProEphemeris

% Asteroid properties.

--------------------------------------------------------------------------
   Form:
   [gm, semi, naif] = AsteroidProperties( name )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   name         (1,:) Asteroid, one of Ceres, Eros, Apophis

   -------
   Outputs
   -------
   gM           (1,1) Gravitational constant (km^3/s^2)
   semi         (1,3) Ellipsoidal semi-major axes (km)
   naif         (1,1) NAIF code

--------------------------------------------------------------------------
