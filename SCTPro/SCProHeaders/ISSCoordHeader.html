ISSCoord:

Path: SCPro/MannedVehicle

% Computes the coordinates in the ISS frame
 The ISS frame is defined as:

   +X Along the velocity vector (Forward)
   -X Opposite the velocity vector (Aft)
   +Y -Orbit Normal (Starboard)
   -Y +Orbit Normal (Port)
   +Z Zenith (Deck)
   -Z Nadir (Overhead)

 This is the same as LVLH.  Julian date propagates the initial ISS coordinate
 set. If there are no outputs the function draws a 3D view of the scene using
 the CAD model in ISS.mat.

 Type ISSCoord for a demo for a position offset from the ISS by a few tens of
 meters.
--------------------------------------------------------------------------
   Form:
   rISS = ISSCoord(r,jD)
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   r   (3,:)	 ECI coordinates
   jD  (1,1)  Julian date

   -------
   Outputs
   -------
   rL  (3,:)  Position in ISS Coordinates

--------------------------------------------------------------------------
   References: https://www.nasa.gov/mission_pages/station/multimedia/
               iss_labs_guide.html
------------------------------------------------------------------------
