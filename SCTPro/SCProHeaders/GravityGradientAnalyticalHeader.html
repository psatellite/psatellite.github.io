GravityGradientAnalytical:

Path: SCPro/DynamicalModels

% Analytical eigenvalues for gravity gradient stabilization.
 r is along -z. This is only valid for a circular orbit.

--------------------------------------------------------------------------
   Form:
   [e, a] = GravityGradientAnalytical( n, iXX, iYY, iZZ )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   n                  (1,1) Orbit rate
   iXX                (1,1) Ixx inertia term
   iYY                (1,1) Iyy inertia term
   iZZ                (1,1) Izz inertia term

   -------
   Outputs
   -------
   e                  (6,1) Eigenvalues
   a                  (6,6) State matrix
   

--------------------------------------------------------------------------
