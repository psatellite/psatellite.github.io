AltitudeSH:

Path: SCPro/ProTools

% Computes the altitude over a spherical harmonic terrain map.
 Type AltitudeSH for a demo flying in a polar orbit over the moon.
--------------------------------------------------------------------------
   Form:
   h = AltitudeSH( r, s, c, nH )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   r       (3,:) Position planet fixed (km)
   s       (:,:) Normalized sine coefficients (m)
   c       (:,:) Normalized cosine coefficients (m)
   nH      (1,1) Number of harmonics to use

   -------
   Outputs
   -------
   h       (1,:) Altitude (km)

--------------------------------------------------------------------------
