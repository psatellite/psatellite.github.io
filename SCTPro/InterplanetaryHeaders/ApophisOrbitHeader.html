% Generate elements and orbit state for the asteroid Apophis.
 Utilizes an ephemeris point from 2009.
--------------------------------------------------------------------------
   Form:
   [elA, rA, vA, jD] = ApophisOrbit
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   None

   -------
   Outputs
   -------

   rA  (3,1) Heliocentric position (km)
   vA  (3,1) Heliocentric velocity (km)
   jD  (1,1) Julian day number of epoch
