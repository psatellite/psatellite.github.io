FEarthMoon:

Path: Interplanetary/RHSFunctions

% Gravity acceleration in the earth moon system
 Uses MoonV1 to locate the moon. Point mass model for the Earth and
 perturbation model for the moon using APlanet.
--------------------------------------------------------------------------
   Form:
   xDot = FEarthMoon( x, t, jD, muMoon, muEarth )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x           (6,1)   [r;v]
   t           (1,1)   Time
   jD          (1,1)   Julian date
   muMoon      (1,1)   Gravitational parameter for the moon
   muEarth     (1,1)   Gravitational parameter for the earth

   -------
   Outputs
   -------
   xDot        (6,1)   State derivative

--------------------------------------------------------------------------
