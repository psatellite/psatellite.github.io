CRTBPJacobiConstant:

Path: Interplanetary/LowEnergyManeuver

% Calculate from Jacobi constant from a scaled state vector
 The default three-body problem is the Earth, Moon, and sun.
 Type CRTBPJacobiConstant for a demo with two identical vectors.
--------------------------------------------------------------------------
   Form:
   j = CRTBPJacobiConstant(y,mu)
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   y   (6,n)	State vector in scaled coordinates [r;v]
   mu	(1,1)	CRTBP mass parameter

   -------
   Outputs
   -------
   j   (n,1)	Jacobi constant
  
--------------------------------------------------------------------------
