CRTBPJacobi:

Path: Interplanetary/ThreeBody

% Compute the Jacobi constant for a state in the CRTBP system.
 Centered at the barycenter. 

--------------------------------------------------------------------------
   Form:
   C = CRTBPJacobi( x, mu )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x            (6,1)     The state vector [r;v]
   mu           (1,1)     Non-dimensional mass parameter.
                           Smaller mass divided by total mass of system.

   -------
   Outputs
   -------
   C            (1,1)     Jacobi constant of integration

--------------------------------------------------------------------------
