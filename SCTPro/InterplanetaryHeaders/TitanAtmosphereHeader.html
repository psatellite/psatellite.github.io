TitanAtmosphere:

Path: Interplanetary/Environment

% Titan density model from stored data.
 Type TitanAtmosphere for a demo. The data extends through 1300 km
 altitude and is interpolated beyond that.

 Requires the mat file TitanDensityData.mat
--------------------------------------------------------------------------
   Form:
   TitanAtmosphere( h )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   h          (1,:) Altitude (km)

   -------
   Outputs
   -------
   rho        (1,:) Density (kg/m^3)

--------------------------------------------------------------------------

Children:

Interplanetary: Data/TitanDensityData
Common: Graphics/Plot2D
