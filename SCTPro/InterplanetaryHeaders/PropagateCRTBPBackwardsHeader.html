PropagateCRTBPBackwards:

Path: Interplanetary/LowEnergyManeuver

--------------------------------------------------------------------------
   Propagate an arbitrary CRTBP with origin at the secondary body
   backwards
--------------------------------------------------------------------------
   Form:
   [yvec, tvec] = PropagateCRTBPBackwards(y,t0,tF,mu);
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   y     (6,1) Current state vector [r;v]
   t0    (1,1) Initial time
   tF    (1,1) Final time (positive)
   mu    (1,1)	CRTBP mass parameter

   -------
   Outputs
   -------
   yvec	(6,:) State vector
   tvec	(1,:) Time vector
  
--------------------------------------------------------------------------
