 Type PlanetSim2D for a demo
--------------------------------------------------------------------------
   Form:
   xP = PlanetSim2D(x,t,a,beta)
--------------------------------------------------------------------------

   -------
   Inputs
   -------
   x    (4,1) State vector [r;v]
   t    (1,:) Time vector
   a    (1,1) Acceleration
   beta (1,:) Thrust angle 0 is along +x

   -------
   Outputs
   -------
   xP   (4,:) Propagated trajectory
