RotateOrbit:

Path: OpticalNavigation/Utilities

% Rotate a trajectory to be in the desired plane
 theta rotates in the plane. The original trajectory is in the xy-plane.
 x2S gives the desired angle of the last element of x1.

 Type RotateOrbit for a demo.

--------------------------------------------------------------------------
   Form:
   x1R = RotateOrbit(x1,x2,mu,x2S,pt)
--------------------------------------------------------------------------

   -------
   Inputs
   -------
   x1      (4,:)	State vector [x;y;vX;vY]
   x2      (6,1)	Desired orbit plane
   mu      (1,1) Gravitational constant
   x2S     (6,1)	Desired ending point of x1 in the x2 plane
   pt      (1,:) 'start' or 'end'

   -------
   Outputs
   -------
   x1R     (6,1)	State vector [r;v]
   b       (3,2) Transformation matrix

--------------------------------------------------------------------------
