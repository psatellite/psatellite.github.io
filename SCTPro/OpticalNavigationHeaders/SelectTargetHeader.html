SelectTarget:

Path: OpticalNavigation/Core

% Determines which target to select for optical navigation

--------------------------------------------------------------------------
   Form:
   [target, blocked] = SelectTarget(r1,r2,d)
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   r1	(3,1) Position of target 1 relative to spacecraft
   r1	(3,1) Position of target 2 relative to spacecraft
   d   (.)   Data structure
             .aBody1	(1,1) Radius of target 1
             .aBody2	(1,1) Radius of target 2

   -------
   Outputs
   -------
   target  (1,1) 1 or 2
   blocked (1,1) 1 if farther target is blocked

--------------------------------------------------------------------------

Children:

Common: Graphics/Plot2D
Math: Linear/Mag
Math: Linear/Unit
