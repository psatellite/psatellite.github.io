JD2Array:

Path: Common/Time

% Compute an array of Julian dates from an epoch.

--------------------------------------------------------------------------
   Forms:
   jD = JD2Array( jD0, t )
--------------------------------------------------------------------------

   -----
   Input
   -----
   jD0   (1,1)    Epoch Julian date
   t     (1,:)    Time array (sec)

   ------
   Output
   ------
   jD    (1,:)    Julian date array

--------------------------------------------------------------------------
