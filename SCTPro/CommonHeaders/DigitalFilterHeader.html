% Implement a digital filter.

   Type DigitalFilter for a demo.

--------------------------------------------------------------------------
   Form:
   [xNew, xN, uN] = DigitalFilter( uNew, nN, dN, xN, uN )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   uNew                   New input
   nN           (1,n)     Numerator polynomial
   dN           (1,n)     Denominator polynomial
   uN           (1,n)     Input storage
   xN           (1,n)     Output storage

   -------
   Outputs
   -------
   xNew                   New output
   uN           (1,n)     Input storage
   xN           (1,n)     Output storage
