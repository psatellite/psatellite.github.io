ArrayToLinspace:

Path: Common/General

% Converts an array a s function string.

 [0 1 2 3 4 5] -> 'linspace(0,5,6)'

 Type ArrayToLinspace for a demo

--------------------------------------------------------------------------
   Form:
   s = ArrayToLinspace( x )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x              (1,:)   Array

   -------
   Outputs
   -------
   s              (1,:)   String

--------------------------------------------------------------------------
