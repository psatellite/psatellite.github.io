AU2M:

Path: Common/Transform

% Convert an angle and a unit vector to a matrix.
   The resulting matrix transforms from the unrotated frame to the
   rotated frame

   Type AU2M for a demo.

--------------------------------------------------------------------------
   Form:
   m = AU2M( angle, u )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   angle         (1,1)   Angle
   u             (3,1)   Unit vector

   -------
   Outputs
   -------
   m             (3,3)   Matrix

--------------------------------------------------------------------------
