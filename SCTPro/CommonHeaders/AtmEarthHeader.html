AtmEarth:

Path: Common/Atmosphere

% Earth atmospheric density
 Agrees with the standard atmosphere model but is faster. Uses AtmDens2

 Type AtmEarth for a demo.

--------------------------------------------------------------------------
   Form:
       AtmEarth;   % demo
   d = AtmEarth( h )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   h   (:)     Altitude (km)

   -------
   Outputs
   -------
   d   (.)     Standard atmosphere
                  .temperature
                  .pressure
                  .density
                  .speedOfSound
                  .kinematicViscosity

--------------------------------------------------------------------------
