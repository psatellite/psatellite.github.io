DiskWithHole:

Path: Common/Graphics

% Draw a disk.

 For correct lighting points must be entered counterclockwise.

 Type Disk for a demo.

--------------------------------------------------------------------------
   Form:
   [v, f] = DiskWithHole( rI, rO, n, z )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   r           (1,1) Radius
   n           (1,1) Number of points
   z           (1,1) z

   -------
   Outputs
   -------
   v           (4,3) Vertices
   f           (4,3) Faces

--------------------------------------------------------------------------
