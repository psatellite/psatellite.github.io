MatrixDisplay:

Path: Common/General

% Display a matrix using the format f

 f is a string like '%12.4f'

 Type MatrixDisplay for a demo.

--------------------------------------------------------------------------
   Form:
    MatrixDisplay( m, f )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x              (:,:)  Matrix
   f             (1,:)  Format

   -------
   Outputs
   -------
   Displays to the command line
