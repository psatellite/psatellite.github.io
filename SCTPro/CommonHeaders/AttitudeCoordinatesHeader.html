AttitudeCoordinates:

Path: Common/Graphics

% Draw attitude coordinates.
 You can add these to any 3D plot.
--------------------------------------------------------------------------
   Form:
   AttitudeCoordinates( varargin )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   varargin  {:} Parameter pairs
                 grid       - on/{off}
                 numbers    - {on}/off
                 radius     - default is 1
                 n          - default is 180
                 ticks      - {on}/off
                 axesHandle - {[]} or specify a handle

   -------
   Outputs
   -------
   None

--------------------------------------------------------------------------
