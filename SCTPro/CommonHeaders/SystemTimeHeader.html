SystemTime:

Path: Common/Time

% Provides the system time to the software.
--------------------------------------------------------------------------
   Form:
   [mET, jD] = SystemTime( action, mET, jD )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   action       (1,:) Action to take
   mET          (1,1) Mission elapsed time
   jD           (1,1) Julian Date

   -------
   Outputs
   -------
   mET          (1,1) Mission elapsed time
   jD           (1,1) Julian Date
