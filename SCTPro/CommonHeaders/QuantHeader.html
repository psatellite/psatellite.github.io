Quant:

Path: Common/Control

% Quantizes a matrix using user defined rounding.

--------------------------------------------------------------------------
   Form:
   [xQ, xCounts] = Quant( x, lsb, rType )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x                      Vector to be quantized
   lsb                    Least significant bit
   rType                  Type of rounding ['round','floor','ceil','fix']
   

   -------
   Outputs
   -------
   xQ                     Quantized x
   xCounts                X in numbers of lsbs

--------------------------------------------------------------------------
