Num2Str1Line:

Path: Common/General

% Converts a matrix to a one line string.
--------------------------------------------------------------------------
   Form:
   s = Num2Str1Line( n, d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   n        (:,:)  Number
   d        (1,1)  Number of decimal places

   -------
   Outputs
   -------
   s        (:)    String

--------------------------------------------------------------------------
