GetHeaders:

Path: Common/FileUtils

% Finds all files in the directory d. 
 Recursively searches through
 subdirectories. Extract the headers. The returned data structure:

 d.fileName
 d.header
 d.fileDirectory

--------------------------------------------------------------------------
   Form:
   d = GetHeaders( p )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   p	        (1,:)   Complete pathname to the directory

   -------
   Outputs
   -------
   d         {}      Data structure

--------------------------------------------------------------------------
