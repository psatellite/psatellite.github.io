Path: Common/DemoFuns

% Right-hand-side for a cubic spring.
 t is not used. This is compatible with RK4 and RK45.
--------------------------------------------------------------------------
   Form:
   d = RHSUKF;
   f = RHSUKF( x, t, d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x         (1,1)  Position
   t         (1,1)  Time (not used)
   d         (1,1)  Data structure
                    .w   (1,1) Spring constant
                    .u   (1,1) Input

   -------
   Outputs
   -------
   f         (1,1)  State derivative

--------------------------------------------------------------------------
