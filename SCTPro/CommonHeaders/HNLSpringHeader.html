% Nonlinear estimator measurement spring model
--------------------------------------------------------------------------
   Form:
   hData = HNLSpring;
   [y, h] = HNLSpring( x, t, hData, k )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x         (2,1)   State
   t         (1,1)   Time
   hData     (.)     Data structure
                     .alpha (1,1) coefficient
   k         (1,1)   Not used

   -------
   Outputs
   -------
   y         (:)    Output
   h         (1,1)  Derivative of y with respect to x
