DistanceLabel:

Path: Common/Graphics

% Rescales distances
 Generates a distance label given the maximum value of d and rescales d.
--------------------------------------------------------------------------
   Form:
   [d, c, u] = DistanceLabel( d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   d	(1,:)   Distance (m)

   -------
   Outputs
   -------
   d	(1,:)   Distance ()
   c	(1,:)   Label (mm, cm, m, km, au, light year)
   u	(1,:)   Units of distance

--------------------------------------------------------------------------

Children:

Common: Graphics/Plot2D
