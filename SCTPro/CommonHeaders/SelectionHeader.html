% Selection GUI.
--------------------------------------------------------------------------
   Form:
   [k,item] = Selection( list );
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   list             {:}  Cell array of strings to choose from
   msg              (:)  Message to display in GUI (string)
   name             {:}  Name in title bar of GUI (string)

   -------
   Outputs
   -------
   k                (1)  Index of chosen item
   item             (:)  Chosen item
