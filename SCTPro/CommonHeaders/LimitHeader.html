Limit:

Path: Common/Control

% Limits a matrix. 
   If limit is a scalar then the limits are set at plus or minus the 
   limit.

--------------------------------------------------------------------------
   Form:
   x = Limit( x, limit )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x                      Vector to be quantized
   limit          (2)     Limit values [max,min]
   

   -------
   Outputs
   -------
   x                      x limited to be between the limits.

--------------------------------------------------------------------------
