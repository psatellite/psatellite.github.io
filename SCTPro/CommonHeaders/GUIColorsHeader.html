GUIColors:

Path: Common/GUIs

% GUI for setting interface colors. 
 Saves the values in the file GUIColors.mat.

 Typing GUIColors is the same as GUIColors('initialize')
--------------------------------------------------------------------------
   Form:
   x = GUIColors( action )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   action      (1,:)  Action 'initialize', 'update'
   modifier    (1,1)  Modifier to the action

   -------
   Outputs
   -------
   x           (1,3) Color

--------------------------------------------------------------------------
