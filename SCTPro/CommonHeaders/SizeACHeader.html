   These are the state transition and state input matrices.

   Form:
   notOK = SizeAC( a, c )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   a                   Plant matrix
   c                   Measurement matrix

   -------
   Outputs
   -------
   notOK               Not correct if equal to 0
