DTSToDTA:

Path: Common/Time

% Converts the date time structure to the date time array

 Typing DTSToDTA returns clock time

--------------------------------------------------------------------------
   Form:
   dateTime = DTSToDTA( t )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   t          (1,1)   Date/time structure
                       .year
                       .month                             
                       .day                             
                       .hour                             
                       .minute                             
                       .second
   -------
   Outputs
   -------
   dateTime    (1,6)  [year month day hour minute seconds 

--------------------------------------------------------------------------
