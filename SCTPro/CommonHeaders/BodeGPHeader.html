BodeGP:

Path: Common/Control

% Computes the phase using the Bode gain phase rule.

   Computes the phase using the Bode gain phase rule where

   phase = 90 d|g|/dw|db/20

   This is only true for minimum phase systems

--------------------------------------------------------------------------
   Form:
   p = BodeGP( g, w )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   g         (n,m)  Complex gain
   w         (1,m)  Frequency Points

   -------
   Outputs
   -------
   p                Phase (deg)

--------------------------------------------------------------------------
