RouthArray:

Path: Common/Control

% Computes the Routhian array for a polynomial.

   Type RouthArray for a demo.
--------------------------------------------------------------------------
   Form:
   r = RouthArray( p )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   p         (1,n)  Polynomial

   -------
   Outputs
   -------
   r         (n,:)  Routh array
