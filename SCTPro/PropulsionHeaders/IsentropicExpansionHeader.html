IsentropicExpansion:

Path: Propulsion/Gas

% Computes isentropic expansion. 

   Type IsentropicExpansion for a demo.

   Since version 8.
--------------------------------------------------------------------------
   Form:
   [pR, rhoR, tR] = IsentropicExpansion( M, gamma )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   M             (1,:)   Mach number
   gamma         (1,1)   Ratio of specific heats

   -------
   Outputs
   -------
   pR            (1,1)   p0/p
   rhoR          (1,1)   rho0/rho
   tR            (1,1)   t0/t

--------------------------------------------------------------------------
