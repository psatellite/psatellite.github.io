MassVolH2O2:

Path: Propulsion/Chemical

% H2/O2 mass and volume.
 For H2 + 1/2 O2 = H2O fR = 0.8889. For the SSME fR ranges from 0.6 to
 0.86.

 Type MassVolH2O2 for a demo of the Space Shuttle external tank.
--------------------------------------------------------------------------
   Form:
   [mH2, mO2, vH2, vO2] = MassVolH2O2( m, fR, type )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   m       (1,:) Mass (kg)
   fR      (1,:) Fuel ratio O2/H2
   type     ''   Type string: 'slush', 'liquid'  

   ------
   Outputs
   -------
   mH2     (1,:) Mass H2 (kg)
   mO2     (1,:) Mass O2 (kg)
   vH2     (1,:) Volume H2 (kg)
   vO2     (1,:) Volume O2 (kg)

--------------------------------------------------------------------------
