% Design an electric transfer vehicle.
 Type TVDesign for a demo.
--------------------------------------------------------------------------
   Form:
   x = TVDesign( mPayload, dV, thrust, eff, sR, pR )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   mPayload  (1,1)    Mass of the payload
   dV        (1,1)    Total delta-V capacity (km/sec)
   thrust    (1,1)    Total vehicle thrust (N)
   eff       (1,1)    Efficiency
   sR        (1,1)    Structural Ratio
   pR        (1,1)    Power ratio

   -------
   Outputs
   -------
   x          (.)    Data structure
