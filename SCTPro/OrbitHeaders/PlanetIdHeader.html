PlanetId:

Path: Orbit/LowEnergyManeuver

% Planet id number from name.
 Includes the moon (10). The sun is 0 for the JPL ephemerides.

 Type PlanetId for a demo.

--------------------------------------------------------------------------
   Form:
   id = PlanetId(planet)
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   planet	{:} Planet names (Mercury through Moon)      

   -------
   Outputs
   -------
   id      (1,1)	Planet id
