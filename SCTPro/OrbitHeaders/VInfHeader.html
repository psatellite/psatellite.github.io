Path: Orbit/OrbitMechanics

% Computes the asymptotic velocity for a hyperbola (v infinity)
--------------------------------------------------------------------------
   Form:
   v = VInf( a, mu )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   a                    Semi-major axis (- for a hyperbola)
   mu                   Gravitational parameter (default = 3.98600436e5)

   -------
   Outputs
   -------
   v                    Asymptotic velocity for a hyperbola

--------------------------------------------------------------------------
   References: Wiesel, W. E., Spaceflight Dynamics, McGraw-Hill, 1988,
               pp. 68.
--------------------------------------------------------------------------
