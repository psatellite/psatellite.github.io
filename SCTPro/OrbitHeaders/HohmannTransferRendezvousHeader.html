HohmannTransferRendezvous:

Path: Orbit/OrbitManeuver

% Compute delta-Vs for rendezvous using Hohmann transfers. 
   The inputs are the target and chase spacecraft elements. 
--------------------------------------------------------------------------
   Form:
   maneuver  = HohmannTransferRendezvous( elT, elC, mu, tol, planet )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   elT           (6,1)   Target orbit elements
   elC           (6,1)   Chase orbit elements
   mu            (1,1)   Gravitational parameter
   tol           (1,1)   Tolerance
   planet        (1,:)   Planet name

   -------
   Outputs
   -------
   maneuver      (1,1)    Maneuver data structure
                          .dV1       (1,1) Transfer orbit insertion delta-V
                          .dV2       (1,1) Target orbit insertion delta-V
                          .tStart    (1,1) Start time relative to epoch
                          .tTransfer (1,1) Time in transfer orbit

--------------------------------------------------------------------------
