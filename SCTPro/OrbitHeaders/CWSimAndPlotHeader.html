CWSimAndPlot:

Path: Orbit/Glideslope

% Visualize a set of maneuvers propagated with Clohessy-Wiltshire equations.
 Has a built-in demo using Glideslope. The simulation uses 1000 points.
--------------------------------------------------------------------------
   Form:
   CWSimAndPlot( x0, w, dV, t )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x0                (6,1)    Initial state (LVLH frame)
   w                 (1)      Orbit rate
   dV                (3,n)    Delta-Vs to apply
   t                 (1,n)    Times of the delta-Vs

--------------------------------------------------------------------------
  See also ClohessyWiltshire.
--------------------------------------------------------------------------
