AdjustMeanAnomaly:

Path: Orbit/OrbitMechanics

% Adjusts mean anomaly in an elements vector for a different start time.
 Only M is changed to account for a time difference. Epoch is the day
 for which the elements were generated. For example

   [el,jD0] = ISSOrbit;

 generates current elements from two line elements downloaded from the 
 web. jD0 is the Julian date of epoch.
--------------------------------------------------------------------------
   Form:
   el = AdjustMeanAnomaly(el,jD0,jD)
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   el  (1,6) [a i W w e M]
   jD0 (1,1) Julian date of epoch (days)
   jD  (1,1) Julian date of start (days)

   -------
   Outputs
   -------
   el  (1,6) [a i W w e M]

--------------------------------------------------------------------------
