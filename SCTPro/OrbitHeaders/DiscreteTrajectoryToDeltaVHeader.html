DiscreteTrajectoryToDeltaV:

Path: Orbit/OrbitControl

% Compute accelerations for a point to point trajectory.
--------------------------------------------------------------------------
   Form:
   [dVDT, dVDTR, dV] = DiscreteTrajectoryToDeltaV( p, vMag )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   p            (3,:)     Points
   vMag         (1,1)     Steady velocity

   -------
   Outputs
   -------
   dVDT         (3,:)     Accelerations at each node
   dVDTR        (3,1)     Radial acceleration at each node
   dV           (3,:)     Velocity change at each node

--------------------------------------------------------------------------

Common: Graphics/Plot2D

Math: Linear/Dot
