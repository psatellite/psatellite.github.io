FHel:

Path: Orbit/RHSOrbit

% Orbit equations for InterstellarSim.

   Since version 2.
--------------------------------------------------------------------------
   Form:
   [xDot, thrust, power] = FHel( x, t, d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x            (5,1)     The state vector [x;y;xDot;yDot;m]
   t                      Time
   d            (:)       Data structure

   -------
   Outputs
   -------
   xDot         (5,1)     The derivative of the state vector
   thrust       (1,1)     The thrust
   power        (1,1)     The available power

--------------------------------------------------------------------------
