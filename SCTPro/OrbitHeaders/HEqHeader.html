% Calculates the orbit angular momentum given the input in equinoctial elements

   Since version 1.
--------------------------------------------------------------------------
   Form:
   h = HEq( eq ,mu )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   eq                    Elements vector [a,P1,P2,Q1,Q2,L,K,l,n]
   mu                    Gravitational parameter

   -------
   Outputs
   -------
   h                     Angular momentum
