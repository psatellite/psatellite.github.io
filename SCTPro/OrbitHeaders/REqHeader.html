REq:

Path: Orbit/OrbitCoord

% Computes the orbit radius and the parameter from equinoctial elements.

   Since version 1.
--------------------------------------------------------------------------
   Form:
   [r, p] = REq( eq )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   eq               Elements vector [a,P1,P2,Q1,Q2,L]

   -------
   Outputs
   -------
   r	               Radius
   p                Parameter

--------------------------------------------------------------------------
 References: Battin, R.H., An Introduction to the Methods and Mathematics
             of Astrodynamics, AIAA, New York, 1987, pp. 484-490.
