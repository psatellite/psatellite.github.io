RLagrange:

Path: Orbit/OrbitCoord

% Computes a collinear Lagrange point. 
 Default is for the earth/moon system.
--------------------------------------------------------------------------
   Form:
   rL = RLagrange( rS, mR, lPoint )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   rS          (3,1)  R of the secondary
   mR          (1,1)  Mass secondary/mass primary
   lPoint      (1,1)  Desired Lagrange point

   -------
   Outputs
   -------
   rL          (3,1)  Lagrange coordinates

--------------------------------------------------------------------------
