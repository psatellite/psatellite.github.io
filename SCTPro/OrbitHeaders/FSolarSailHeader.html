FSolarSail:

Path: Orbit/RHSOrbit

% Solar sail simulation right hand side.
--------------------------------------------------------------------------
   Form:
   vargout = FSolarSail( t, x, flag, d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   t                      Time
   x             (6,1)    The state vector [r;v]
   flag          (1,:)    Flags for ode113
   d             (1,1)    Data structure
                          .jDStart   (1,1) Julian date of simulation start
                          .muSun     (1,1) Central body gravitational parameter
                          .sail      (1,1) Sail data structure
                                           .normal (3,1) Sail normal
                                           .area   (1,1) Sail area
                                           .sigmaT (1,1) Sail transparency
                                           .sigmaA (1,1) Sail absorptivity
                                           .sigmaD (1,1) Sail diffusivity
                                           .sigmaS (1,1) Sail specular reflection
                          .mass      (1,1) Spacecraft mass
                          .planetsOn (1,1) 1 if planets are on

   -------
   Outputs
   -------
   xDot          (6,1)     The state vector derivative d[r;v]/dt

--------------------------------------------------------------------------
