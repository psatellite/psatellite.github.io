TrueAnomalySpacing:

Path: Orbit/OrbitCoord

% Generates an array of true anomalies. 

--------------------------------------------------------------------------
   Form:
   [nu, dT, t] = TrueAnomalySpacing( e, nu0, n, duration, nS, eTol )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   e           (1,1)   Eccentricity
   nu0         (1,1)   Initial true anomaly (rad)
   n           (1,1)   Mean orbit rate (rad/s)
   duration    (1,1)   Duration of array (sec)
   nS          (1,1)   Total number of samples
   eTol        (1,1)   Tolerance on e (circular orbit)

   -------
   Outputs
   -------
   nu          (1,:)	True anomalies
   dT          (1,:)	Time steps
   t           (1,:+1)	Time vector

--------------------------------------------------------------------------
