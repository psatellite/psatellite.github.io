REl:

Path: Orbit/OrbitCoord

% Computes the orbit radius and the parameter from Keplerian elements.

   Since version 1.
--------------------------------------------------------------------------
   Form:
   [r, p] = REl( el )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   el                  Elements vector [a,i,W,w,e,M] where a is perigee
                       radius for a parabola

   -------
   Outputs
   -------
   r	                  Radius
   p                   Parameter

--------------------------------------------------------------------------
