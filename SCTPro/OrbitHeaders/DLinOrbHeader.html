DLinOrb:

Path: Orbit/RHSOrbit

% Computes the linearized orbit equations for a discrete system. 

 x[k+1] = ad*x[k] + bd*u[k]

--------------------------------------------------------------------------
   Form:
   [ad,bd] = DLinOrb( n, dT )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   n                (1,1) mean orbit rate (rad/s)
   dT               (1,1) time step

   -------
   Outputs
   -------
   ad               (6,6) State transition matrix 
   bd               (6,3) Input matrix

--------------------------------------------------------------------------
