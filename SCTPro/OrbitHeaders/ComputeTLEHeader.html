% Compute two-line elements from osculating elements
--------------------------------------------------------------------------
   Form:
   sTLE = ComputeTLE( elOsc, jD, Bstar )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   elOsc        (1,6)   Osculating elements [a,i,W,w,e,M]
   jD           (1,1)   Julian date
   Bstar        (1,1)   Bstar drag parameter

   -------
   Outputs
   -------
   sTLE         (3,:)   Two-line elements string

--------------------------------------------------------------------------
   Reference:
   NORAD TLE CONVERSION FROM OSCULATING ORBITAL ELEMENT, Byoung-Sun Lee, J.
   Astron. Space Sci. 19(4), 395-402 (2002)
