ROrb:

Path: Orbit/OrbitCoord

% Computes the orbit radius.

 Type ROrb for a demo
--------------------------------------------------------------------------
   Form:
   r = ROrb( e, rp, f )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   e   (1,1) Eccentricity
   rp	(1,1) Perigee radius
   f   (1,:) True anomaly

   -------
   Outputs
   -------
   r   (1,:) Radius

--------------------------------------------------------------------------
   References: Bate, R. R., Fundamentals of Astrodynamics. pp. 20-40.
--------------------------------------------------------------------------
