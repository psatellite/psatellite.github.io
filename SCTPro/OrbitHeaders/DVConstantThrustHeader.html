DVConstantThrust:

Path: Orbit/StraightLine

% Velocity change with constant thrust given fuel and time
 Computes the velocity change with constant thrust as a function of exhaust
 velocity. This is an analytical solution. If the time vector is not entered,
 it consumes all of the fuel.

 Type DVConstantThrust for a demo.
--------------------------------------------------------------------------
   Form:
   dV = DVConstantThrust( thrust, mD, mF, uE, t )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   thrust  (1,1) Thrust level (N)
   mD      (1,1) Dry mass (kg)
   mF      (1,1) Fuel mass (kg)
   uE      (1,:) Exhaust velocity (km/s)
   t       (1,:) Time (s)

   -------
   Outputs
   -------
   dV      (1,:) Velocity change (m/s)

--------------------------------------------------------------------------
