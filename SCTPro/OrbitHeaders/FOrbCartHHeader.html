FOrbCartH:

Path: Orbit/RHSOrbit

% Computes the right-hand-side of a spherical harmonic orbit (Cartesian). 
 Includes an external acceleration.
--------------------------------------------------------------------------
   Forms:
   xDot = FOrbCartH( x, t, accel, gravityModel, nN, nM, jD )
   xDot = FOrbCartH( t, x, accel, gravityModel, nN, nM, jD )  % ode113 format
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x            (6,1)     The ECI state vector [r;v]
   t            (1,1)     Time (unused)
   accel        (3,1)     External acceleration (default is zero)
   gravityModel  (:)      Gravitational model (no default)
   nN           (1,1)     Highest zonal harmonic (2)
   nM           (1,1)     Highest sectorial and tesseral harmonic (0)
   jD           (1,1)     Julian Date

   -------
   Outputs
   -------
   xDot         (6,1)     The derivative of the state vector

--------------------------------------------------------------------------
   See also: ECIToEF, AGravityC, RHSGeoHarm
--------------------------------------------------------------------------
