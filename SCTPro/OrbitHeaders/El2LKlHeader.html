El2LKl:

Path: Orbit/OrbitCoord

% Converts orbital elements to commonly used angular quantities

   Since version 1.
--------------------------------------------------------------------------
   Form:
   [L, K, l] = El2LKl( el )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   el                    Elements vector [a,i,W,w,e,M]

   -------
   Outputs
   -------
   L                     True longitude      ( w + W + f )
   K                     Eccentric longitude ( w + W + E )
   l                     Mean longitude      ( w + W + M )

--------------------------------------------------------------------------
