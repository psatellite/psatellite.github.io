J2Prop:

Path: Orbit/OrbitMechanics

% Propagate an ECI state into the future with the J2 perturbation.
 Uses FOrbCartH with the GEMT1.geo Earth gravity model, specifying only
 the second zonal harmonic, nN = 2, nM = 0. 
--------------------------------------------------------------------------
   Form:
   x = J2Prop( x0, jD, time, gM, showWaitBar )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x0              (6,1) Initial state [km; km/s]
   jD              (1,1) Initial Julian Date [JD]
   time            (1,:) Time vector [sec]
   gM              (.)   Gravity Model
   showWaitBar     (1,1)  Flag. Set to 1 to show wait bar. Default 0.

   -------
   Outputs
   -------
   x               (6,:) Time history of ECI states [km; km/s]

--------------------------------------------------------------------------
