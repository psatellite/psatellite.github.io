MEqToRV:

Path: Orbit/OrbitCoord

% Transforms modified equinoctial elements to r and v.  
   The modified equinoctial elements are

   p = a(1 + e^2)
   f = ecos(w + W);
   g = esin(w + W);
   h = tan(i/2)cosW
   k = tan(i/2)sinW
   L = W + w + nu

   W is right ascension of the ascending node
   w is argument of perigee

   Since version 7.
--------------------------------------------------------------------------
   Form:
   [r, v] = MEqToRV( x, mu )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x              (6,:)  Elements
   mu             (1,:)  Gravitational parameter

   -------
   Outputs
   -------
   r              (3,:) Position vector
   v              (3,:) Velocity vector

--------------------------------------------------------------------------
