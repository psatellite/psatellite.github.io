--------------------------------------------------------------------------
   Form:
   b = ChangePerigee( d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   d			  (:)     Input data structure
                         .a   (1,1)   Semi-major axis
                         .e   (1,1)   Eccentricity of initial orbit
                         .rP  (1,:)   New Perigee
                         .mu  (1,1)   Mu
   
                        
   -------
   Outputs
   -------
   b             (:)     Output data structure
                         .dV  (1,:)   Datastructure
                         .e   (1,:)   New eccentricity
                         .a   (1,:)   New semi-major axis
                         .vA  (1,:)   Apogee velocity
                        
--------------------------------------------------------------------------
