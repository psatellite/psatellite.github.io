MajorMinorToAE:

Path: Orbit/OrbitMechanics

% Convert semi-major and semi-minor lengths to a and e
 
 Type MajorMinorToAE( sMin, sMaj ) for a demo.

--------------------------------------------------------------------------
   Form:
   [a,e] = MajorMinorToAE( sMin, sMaj )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   sMin (1,:) Semi-minor axis
   sMaj (1,:) Semi-major axis

   -------
   Outputs
   -------
   sMaj (1,:) Semi-major
   e    (1,:) Eccentricity

--------------------------------------------------------------------------
