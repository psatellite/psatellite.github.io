% Time vector for a TLE

 Time vector for a TLE

 Type TLETime for a demo.

--------------------------------------------------------------------------
   Form:
   t = TLETime( tLE, startDate, endDate, n )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   tLE         (1,:)           TLE string
   startDate   (1,1) or (1,6)  Julian date or date array
   endDate     (1,1) or (1,6)  Julian date or date array
   n           (1,1)           Number of steps (default is 100)

   -------
   Outputs
   -------
   t           (1,n)  Time vector
