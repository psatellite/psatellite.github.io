SynodicPlanets:

Path: Orbit/Interplanetary

% Computes the synodic period from planets.

   Type SynodicPlanets for an example in Jupiter orbit.

   Since version 2014.1
--------------------------------------------------------------------------
   Form:
    p = SynodicPlanets( planet1, planet2, jD )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   planet1   (1,:)   Starting planet name
   planet2   (1,:)   Target planet name
   jD        (1,1)   Julian date

   -------
   Outputs
   -------
   p        (1,1)   Synodic period

--------------------------------------------------------------------------
   See also Planets, SolarSystemElements, SynodicFromSemimajorAxes
--------------------------------------------------------------------------
