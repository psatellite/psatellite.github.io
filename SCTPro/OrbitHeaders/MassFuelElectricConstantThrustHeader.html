MassFuelElectricConstantThrust:

Path: Orbit/StraightLine

% Fuel masss for a constant thrust electric propulsion system.
 Assumes constant exhaust velocity and constant thrust. Flyby and stopping are
 supported. This uses a straight line model.

 Type MassFuelElectricConstantThrust for a demo.
--------------------------------------------------------------------------
   Form:
   d = MassFuelElectricConstantThrust( d0 )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   d0    	(.)   Data structure
                 See StraightLineDataStructure

   -------
   Outputs
   -------
   d     	(.)   Data structure with updated fields
                 See StraightLineDataStructure
