LRLoopAntenna:

Path: Electrical/Components

% Computes the inductance and resistance of a circular loop antenna

 Default: copper. 

 Type LRLoopAntenna for a demo.

--------------------------------------------------------------------------
   Form:
   [l, r, rLoss, rRad] = LRLoopAntenna( dAnt, dWire, f, n, rho )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   dAnt 	(1,1) Antenna diameter (meters)
   dWire (1,1) Wire diameter (meters)
   f     (1,:) Frequency (Hz)
   n     (1,1) Number of loops
   rho   (1,1) Resistivity (Ohm*m, default Copper)

   -------
   Outputs
   -------
   l    	(1,1)	Inductance (H)
   r     (1,:) Total resistance (Ohm)
   rLoss	(1,:) Resistance loss (Ohm)
   rRad	(1,:) Radiation loss (Ohm)

--------------------------------------------------------------------------
   Reference:  https://www.ece.mcmaster.ca/faculty/nikolova/antenna_dload/current_lectures/L12_Loop.pdf
               https://www.eeweb.com/tools/loop-inductance
--------------------------------------------------------------------------
