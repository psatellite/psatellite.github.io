PIRectifier:

Path: Electrical/Components

% Implement proportional integral control of a rectifier

 This works in the direct-quadrature frame. All multi-phase 
 motors or circuits can be transformed to direct and quadrature 
 components.

--------------------------------------------------------------------------
   Form:
        d = PIRectifier;  % data structure
   [uR,d] = PIRectifier(iDQ,iDQStar,d)
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   iDQ     (2,1) [direct;quadrature] current (A)
   iDQStar (2,1) [direct;quadrature] current reference (A)
   d       (.)   Data structure
                 .L              (1,1) Inductance (H)
                 .omega          (1,1) Electrical angular rate (rad/s)
                 .dT             (1,1) Time step (s)
                 .integralIDQErr (1,1) Integral of the error
                 .gainDQ         (1,1) Forward gain
                 .tauDQ          (1,1) Time constant (s)

   -------
   Outputs
   -------
   uRDQ     (2,1) [direct;quadrature] voltage (V)
   d        (.)   Updated data structure
                  .integralIDQErr (1,1) Integral of the error
