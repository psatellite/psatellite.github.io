NPhase:

Path: Electrical/Components

% Generate an n phase signal.

   The magnitude is +/- 1.

   Type NPhase for a demo.

--------------------------------------------------------------------------
   Form:
   v = NPhase( w0, t, n )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   w0           (1,1) Base frequency (rad/s)
   t            (1,:) Time
   n            (1,1) Number of phases

   -------
   Outputs
   -------
   v            (n,:) Phases

--------------------------------------------------------------------------
