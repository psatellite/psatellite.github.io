ResistanceAWG:

Path: Electrical/Wire

% Resistance from AWG gauge.
   Default conductivity is for copper.
------------------------------------------------------------------------
   Form:
       ResistanceAWG;
   r = ResistanceAWG( n, sigma )
------------------------------------------------------------------------

   ------
   Inputs
   ------
   n           (1,:) AWG gauge 0000 is -3
   sigma       (1,1) Conductivity (1/Ohm*m)

   -------
   Outputs
   -------
   r           (1,:) Resistance (Ohm/m)
   
------------------------------------------------------------------------
