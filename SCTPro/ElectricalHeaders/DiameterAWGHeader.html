Path: Electrical/Wire

% Diameter from AWG gauge.
 The gauge is a number from -3 to 36.
------------------------------------------------------------------------
   Form:
       DiameterAWG;
   d = DiameterAWG( n )
------------------------------------------------------------------------

   ------
   Inputs
   ------
   n          (1,:) AWG gauge 0000 is -3

   -------
   Outputs
   -------
   d          (1,:) Diameter (mm)
   

------------------------------------------------------------------------
   Reference:
   http://en.wikipedia.org/wiki/American_wire_gauge#
   Table_of_AWG_wire_sizes
------------------------------------------------------------------------
