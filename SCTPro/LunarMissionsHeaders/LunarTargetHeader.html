--------------------------------------------------------------------------
   Form:
   r = LunarTarget( x, jD, downrange )
--------------------------------------------------------------------------

   -------
   Inputs

   x         (20,1)  State vector [r;v;..] (km,km/s)

   downrange (3,1)   Downrange distance

   -------
   Outputs
   -------
   r         (3,1)   Landing spot in the lunar fixed frame
