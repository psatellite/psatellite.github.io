LandingLegsCAD:

Path: LunarMissions/LunarMission

% Generates a CAD component for a lunar module first stage legs.
 Use 'scale' to change the size of the legs. Use 'xy spacing' to adjust the
 distance between the legs. These legs are based on the Apollo Lunar Module.

 Type LandingLegsCAD for a demo.

--------------------------------------------------------------------------
   Form:
   [v, f] = LandingLegsCAD( varargin )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   'scale' and 'xy spacing'

   -------
   Outputs
   -------
   v(:,3) Vertices
   f(:,3) Faces
   a(1,:) Area

--------------------------------------------------------------------------
