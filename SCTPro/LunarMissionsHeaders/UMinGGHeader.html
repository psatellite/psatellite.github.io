UMinGG:

Path: LunarMissions/Gateway

% Vector from ECI to body that minimizes the gravity gradient torque
 Uses fminsearch. Type UMinGG for a demo with NASA Gateway.
--------------------------------------------------------------------------
   Form:
   u = UMinGG(inr)
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   inr           (3,3)  Inertia matrix

   -------
   Outputs
   -------
   u             (3,1)  Unit vector

--------------------------------------------------------------------------
