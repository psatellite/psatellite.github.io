BTrans:

Path: SC/GOESRead

% Transforms the GOES data into popular earth pointing frames.

   Since version 3.
--------------------------------------------------------------------------
   Form:
   bT = BTrans( b, aS )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   b                (3,:)   GOES Field Data
   aS               (1,:)   Axis system:
                              'GEOC'  z North, x Zenith
                              'NASA'  y South  z Nadir  
                              Default is 'NASA'                           
   

   -------
   Outputs
   -------
   bT               (3,:)   Transformed data

--------------------------------------------------------------------------
