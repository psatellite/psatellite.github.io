GOESLon:

Path: SC/GOESRead

% Computes the west longitude of a GOES spacecraft.
   See also LoadSatP.

   Since version 3.
--------------------------------------------------------------------------
   Form:
   l = GOESLon( sc, dateTime, file )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   sc               (1,1)   GOES spacecraft ID
   dateTime         (1,6)   [year month day hour minute second]
   file             (1,:)   File name for position data
   

   -------
   Outputs
   -------
   l                (1,1)   Longitude

--------------------------------------------------------------------------
