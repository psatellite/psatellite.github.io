Glitch:

Path: SC/GOESRead

% Removes glitches from data. 
 If bMax is omitted it will look for  zeros. It linearly interpolates gaps with
 glitch data using the valid values on the edges of the glitch regions.
--------------------------------------------------------------------------
   Form:
   bD = Glitch( b, bMax, ifLT )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   b                (:,:)   Data to be deglitched
   bMax             (1,1)   Threshold indicating bad data   > bMax
   ifLT             (1,1)   Any argument will have it check < bMax
   

   -------
   Outputs
   -------
   bD               (:,:)   Deglitched data

--------------------------------------------------------------------------
