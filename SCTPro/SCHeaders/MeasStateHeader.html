MeasState:

Path: SC/Sensor

% Returns elements of a state vector as a measurement.
   You enter a scale factor and a bias in the data structure. 
   The noise field should be set to zero when this is used a part of a
   Kalman filter.

   The field "k" selects elements of the state vector x.
   This is useful for initial tests of control algorithms or estimators.

   The state vector can be any state vector of interest.

   If no inputs are given it will return the default data structure.

   Since version 11.
--------------------------------------------------------------------------
   Form:
   y = MeasState( x, d )
   d = MeasState
--------------------------------------------------------------------------

   ------
   Inputs    
   ------
   x         (n,1)  State vector
   d         (1,1)  Data structure
                    .k     (1,m) States to return
                    .scale (m,1) Scale factor
                    .bias  (m,1) Bias
                    .noise (m,1) 1 sigma noise

   -------
   Outputs
   -------
   y         (m,1)   Measurements

--------------------------------------------------------------------------
