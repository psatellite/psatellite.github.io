MagneticTorquerDipoleMoment:

Path: SC/Actuator

% Compute the moment for a magnetic torquer.
   For an air coil set muR = 1.

   Type MagneticTorquerDipoleMoment for a demo.

   Since version 9.
--------------------------------------------------------------------------
   Form:
   m = MagneticTorquerDipoleMoment( l, r, muR, n, i )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   l           (1,1)   Torquer length (m)
   r           (1,1)   Torquer radius (m)
   muR         (1,1)   Relative permeability
   n           (1,1)   Number of turns of wire
   i           (1,1)   Current (A)

   -------
   Outputs
   -------
   m           (1,1)   Dipole moment (Amp-turns m^2);

--------------------------------------------------------------------------
   Reference: http://mac6.ma.psu.edu/lionsat/TorquerCDRv5.html
--------------------------------------------------------------------------
