TwoUToQ:

Path: SC/Attitude

% Finds the quaternion that transforms u1 and u2 to u1T and u2T

 Type TwoUToQ for a demo of a spacecraft in LEO using the magnetic field
 and nadir vectors for a spacecraft in LVLH.

--------------------------------------------------------------------------
   Form:
   q = TwoUToQ(u1,u2,u1T,u2T)
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   u1  (3,:) Unit vector 1
   u2  (3,:) Unit vector 2
   u1T (3,:) Unit vector 1 transformed
   u2T (3,:) Unit vector 2 transformed

   -------
   Outputs
   -------
   q   (4,:)	Quaternion from the u1/u2 frame to the u1T/u2T frame

--------------------------------------------------------------------------
   See also U2Q.m
--------------------------------------------------------------------------
