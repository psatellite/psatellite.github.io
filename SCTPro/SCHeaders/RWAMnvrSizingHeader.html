RWAMnvrSizing:

Path: SC/Actuator

% Size a reaction wheel using maneuver requirements.
 This uses a bang-bang maneuver. Either theta or tau may be an array to
 get a plot.

 Type RWAMnvrSizing for a demo of a 3U CubeSat 

--------------------------------------------------------------------------
   Form:
   [t, h] = RWAMnvrSizing( inr, theta, tau )
            RWAMnvrSizing( inr, theta, tau ); % plot or display result
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   inr         (1,1)  Spacecraft inertia (kg-m^2)
   theta       (1,1)  Maneuver angle (rad)
   tau         (1,:)  Maneuver duration (s)

   -------
   Outputs
   -------
   t           (1,:)  Torque (Nm)
   h           (1,:)  Momentum storage (Nms)

--------------------------------------------------------------------------
