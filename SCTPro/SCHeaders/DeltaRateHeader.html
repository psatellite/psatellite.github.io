DeltaRate:

Path: SC/Attitude

% Computes the new angular rate when the inertia changes.
--------------------------------------------------------------------------
   Form:
   omega = DeltaRate( InertiaOld, InertiaNew, omega )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   inertiaOld     (3,3)  Old inertia
   inertiaOld     (3,3)  New inertia
   omega          (3,1)  Angular rate
 
   -------
   Outputs
   -------
   omegaNew       (3,1)  Angular rate

--------------------------------------------------------------------------
