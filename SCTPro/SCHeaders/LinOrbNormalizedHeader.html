LinOrbNormalized:

Path: SC/Dynamics

% Computes the normalized linearized orbit state equations.

   y      = c*x[k] + d*u[k]
   x[k+1] = a*x[k] + b*u[k]

   otherwise

   y      = c*x + d*u
   dx/dt  = a*x + b*u

--------------------------------------------------------------------------
   Form:
   [a, b, c, d] = LinOrbNormalized
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   None

   -------
   Outputs
   -------
   a                (6,6) State transition matrix or (6,1) state derivative
   b                (6,3) Input matrix
   c                (3,6) Output matrix (position)
   d                (3,3) Feedthrough matrix
