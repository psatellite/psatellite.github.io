GravityGradientEigenvalues:

Path: SC/Dynamics

% Compute the eigenvalues for a gravity gradient stabilized spacecraft.
   We assume it is in the LVLH orientation.

--------------------------------------------------------------------------
   Form:
   [e, a] = GravityGradientEigenvalues( inr, r, v, mu )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   inr                (3,3) Inertia matrix
   r                  (3,1) r in ECI
   v                  (3,1) v in ECI
   mu                 (1,1) Gravitational Parameter

   -------
   Outputs
   -------
   e                  (6,1) Eigenvalues
   a                  (6,6) State matrix
   

--------------------------------------------------------------------------
