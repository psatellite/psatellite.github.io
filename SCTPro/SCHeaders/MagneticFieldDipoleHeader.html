MagneticFieldDipole:

Path: SC/Actuator

% Computes the magnetic field of a dipole
   Calculates the magnetic field for a dipole at the origin

   Type MagneticFieldDipole for a demo.
--------------------------------------------------------------------------
   Form:
   b = MagneticFieldDipole( m, r )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   m	(3,1)   Dipole vector (ATM^2)
   r	(3,:)   Vector to the points

   -------
   Outputs
   -------
   b	(3,:)   Magnetic field (T)

--------------------------------------------------------------------------
