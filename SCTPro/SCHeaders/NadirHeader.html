Nadir:

Path: SC/Sensor

% Computes nadir angle from a cant angle, earth angle and chordwidth

   Since version 2.
--------------------------------------------------------------------------
   Form:
   n = Nadir( cant, rho, chordwidth )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   cant       (1,1)   Sensor element cant angle from the spin-axis (rad)
   rho        (1,1)   Earth radius
   chordwidth (:)     Measured chordwidth (rad)

   -------
   Outputs
   -------
   n          (2,:)   Nadir angles (rad)
   k          (1,:)   Valid chordwidths cooresponding to n

--------------------------------------------------------------------------
