RHSPlanetTakeoff:

Path: SC/Dynamics

% Right-hand-side for a 2 dimensional planet takeoff.
 This function is compatible with RK2, RK4 and RK45.
--------------------------------------------------------------------------
   Form:
   xDot = RHSPlanetTakeoff( x, ~, a, g, beta, signX )
--------------------------------------------------------------------------

   -------
   Inputs
   -------
   x       (4,1)	State vector [x;y;vX;vY]
   t       (1,1)	Time (s) (unused)
   a       (1,1)	Thruster acceleration
   g       (1,1) Gravitational acceleration along -y
   beta    (1,1) Thrust angle from the horizontal (rad)
   signX   (1,1) Account for desired x acceleration direction (default 1)

   -------
   Outputs
   -------
   xDot	  (:,1) d[x;y;vX;vY]/dt

--------------------------------------------------------------------------
