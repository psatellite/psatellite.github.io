% Linearized attitude dynamics for a rigid body.

--------------------------------------------------------------------------
   Form: 
   d = RHSRigidBodyLinear
   xDot = RHSRigidBodyLinear( x, t, d )
--------------------------------------------------------------------------

   -------
   Inputs
   -------
   x       (7,1) State vector [q;omega]

   d       (1,1) Data structure

                   .torque     (3,1) Torque vector

   Outputs
   -------
   xDot	(:,1)	d[q;omega]/dt
