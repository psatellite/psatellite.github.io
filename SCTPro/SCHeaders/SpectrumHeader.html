Spectrum:

Path: SC/Sensor

% Gives the spectrum for radiation using conventional names.
   If you enter no inputs it will give you a table of the names and wavelength 
   ranges.
--------------------------------------------------------------------------
   Form:
   lambda = Spectrum( band, n )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   band           (1,;) Name of band
   n              (1,1) Number of elements, default is 100

   -------
   Outputs
   -------
   lambda         (1,:)  Wavelengths

--------------------------------------------------------------------------
