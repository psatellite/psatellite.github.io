GravityGradientStiffness:

Path: SC/Disturbances

%   Computes gravity gradient stiffness from inertia and orbit rate.
   This is equivalent to the spring constant.
--------------------------------------------------------------------------
   Form:
   s = GravityGradientStiffness( inertia, orbitRate )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   inertia    (3,3)  Inertia matrix
   orbitRate  (1,1)  Orbit rate

   -------
   Outputs
   -------
   s          (2,1)  Stiffness [roll,pitch] (Nm/rad)

--------------------------------------------------------------------------
   See also FGravityGradientStiffness
--------------------------------------------------------------------------
