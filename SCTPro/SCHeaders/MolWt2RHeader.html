MolWt2R:

Path: SC/Environs

% Computes R given the molecular weight.
   Type MolWt2R for a demo.

   Since version 1.
--------------------------------------------------------------------------
   Form:
   r = MolWt2R( molWt )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   molWt      (1,:)   Molecular weight (kg/mole). This can be read
                      directly from a periodic table. For example
                      H2 has a molecular weight of 0.002. He has a 
                      molecular weight of 0.004.

   -------
   Outputs
   -------
   r          (1,:)   Specific gas constant 

--------------------------------------------------------------------------

Common: Graphics/Plot2D
