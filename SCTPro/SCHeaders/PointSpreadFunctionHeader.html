PointSpreadFunction:

Path: SC/Sensor

% Point spread function for a circular aperture
 Computes the point spread function as a function of u for a circular aperture
 where u is kar/R and a is the aperture radius, k is the wave number
 (2pi/lambda) and R is the distance between the projection and the aperture.

--------------------------------------------------------------------------
   Form:
   i = PointSpreadFunction( u )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   u         (:)     Argument

   -------
   Outputs
   -------
   i		      (:)     Intensity

--------------------------------------------------------------------------
   References:	Hecht, E. Optics, p. 419.
--------------------------------------------------------------------------
