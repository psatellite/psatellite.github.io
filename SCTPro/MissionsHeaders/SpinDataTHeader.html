SpinDataT:

Path: Missions/Comsat

% Generates the data needed for a spin change in TrnsfOrb.m

--------------------------------------------------------------------------
   Form:
   x = SpinDataT( axis, wF, tID, dTDelay )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   axis         (3,1)  Axis for spin change
   wF           (1,1)  Final spin rate after each burn (up to 3) (rpm)
   tID          (1,n) Thrusters to be used
   dTDelay      (1,1) Delay between burns (1 less than the # of burns)

   -------
   Outputs
   -------
   x            (1,7+n) Formatted data

--------------------------------------------------------------------------
