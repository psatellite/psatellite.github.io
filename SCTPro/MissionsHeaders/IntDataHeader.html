% Generates the data needed for initialization in TransferOrbit.m
--------------------------------------------------------------------------
   Form:
   x = IntData( elements, rA, dec, w, hW, fuelMass )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   elements     (1,6)    Keplerian elements
   rA           (1,1)    Right ascension (deg)
   dec          (1,1)    Declination (deg)
   w            (3,1)    Spin vector (rad/sec)
   hW           (1,1)    Wheel momentum
   fuelMass     (1,n)    Fuel mass (per tank) (lbf)

   -------
   Outputs
   -------
   x            (1,8+n)  Formatted data
