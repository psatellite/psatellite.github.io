Path: Missions/Comsat

% Computes the thrust and fuel consumed for thrusters. 
 The units in this model will depend on the units in the database. 
 p must correspond to the database units. 

--------------------------------------------------------------------------
   Form:
   [thrust, dF, impulse, iSP] = REA( id, p, dT, database )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   id              (:)    Thruster ids
   p               (:)    Pressure
   dT              (:)    Pulsewidth (sec)
   database        (1,:)  Database name

   -------
   Outputs
   -------
   thrust          (:)    Thrust
   dF              (:)    Fuel consumed
   impulse         (:)    Total impulse per thruster 
   iSp             (:)    Specific impulse

--------------------------------------------------------------------------

Children:

Math: Linear/SumC
