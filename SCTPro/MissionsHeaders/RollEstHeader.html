RollEst:

Path: Missions/Comsat

% Designs the dual-spin-turn roll estimator

   Since version 1.
--------------------------------------------------------------------------
   Form:
   [L, a] = RollEst( inr, hW, wo, q, r, dT, sType )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   inr             (3,3)  Inertia
   hW              (1,1)  Internal momentum
   wo              (1,1)  Spin rate
   q               (2,2)  State noise
   r               (1,1)  Measurement noise
   dT              (1,1)  Time step
   sType           (1,:)  Type of transform

   -------
   Outputs
   -------
   L               (2,1)  Gain matrix
   a               (2,2)  State transition matrix

--------------------------------------------------------------------------
