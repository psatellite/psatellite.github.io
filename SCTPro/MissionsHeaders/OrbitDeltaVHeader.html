OrbitDeltaV:

Path: Missions/Comsat

% Updates orbital elements based on an impulsive delta-V
--------------------------------------------------------------------------
   Form:
   el = OrbitDeltaV( el, dV, mu )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   el              (1,6)  Keplerian Elements [a,i,W,w,e,M] (angles in rad)
   dV              (3,1)  Vector delta-V
   mu              (1,1)  Gravitational constant

   -------
   Outputs
   -------
   el              (1,6)  Keplerian Elements [a,i,W,w,e,M] (angles in rad)

--------------------------------------------------------------------------
