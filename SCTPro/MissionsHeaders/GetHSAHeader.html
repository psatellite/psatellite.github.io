GetHSA:

Path: Missions/Comsat

% Gets HSA attitude data

   Since version 1.
--------------------------------------------------------------------------
   Form:
   GetHSA( w, rA, dec, el, jD0, dT, nPts, database )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   w               (3,1)  Body rate
   rA              (1,1)  Right ascension (rad)
   dec             (1,1)  Declination (rad)
   el              (1,6)  Keplerian Elements [a,i,W,w,e,M]
   jD0             (1,1)  Initial Julian date
   dT              (1,1)  Time between samples (sec)
   nPts            (1,1)  How many samples
   database        (1,:)  Spacecraft properties database

   -------
   Outputs
   -------
   None              

--------------------------------------------------------------------------
