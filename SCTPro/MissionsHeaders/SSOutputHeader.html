SSOutput:

Path: Missions/SunNadir

% Models the output for array mounted analog sun sensors.

   Since version 1.
--------------------------------------------------------------------------
   Form:
   m = SSOutput( e, nE, u, uSun, a, n )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   e            (2,1)     Solar array angles
   nE           (4,1)     Index for the sun sensors that tells which array
                          they are on (1 or 2)
   u            (3,4)     Sun sensor eye vectors
   uSun         (3,1)     The sun vector in the body frame
   a            (:,1)     Eye coefficients
   n            (1,1)     Number of counts above zero (e.g. for an 8 bit output
                          enter 256) 
   

   -------
   Outputs
   -------
   m            (1,4)     Eye output in counts

--------------------------------------------------------------------------
