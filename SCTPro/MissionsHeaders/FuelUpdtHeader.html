FuelUpdt:

Path: Missions/Comsat

% Updates the fuel mass.
   This function can handle any number of thrusters and any number
   of fuel tanks (i.e. elements of fuelMass.)

   Since version 1.
--------------------------------------------------------------------------
   Form:
   fuelMass = FuelUpdt( fuelUsed, tId, fuelMass, database )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   fuelUsed     (1,n)     Fuel used by thrusters
   tId          (1,n)     Thrusters used
   fuelMass     (1,m)     Fuel mass
   database     (1,:)     Spacecraft properties database

   -------
   Outputs
   -------
   fuelMass     (1,m)     Fuel mass

--------------------------------------------------------------------------
