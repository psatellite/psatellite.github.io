OddEven:

Path: Missions/Comsat

% Count the number of odd and even numbered thrusters in use. 

   Since version 1.
--------------------------------------------------------------------------
   Form:
   [nOdd, nEven] = OddEven( thrusterIDs )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   thrusterIDs  (n)   Array of the thruster ID numbers

   -------
   Outputs
   -------
   nOdd         (1,1) Number of odd thrusters
   nEven        (1,1) Number of even thrusters

--------------------------------------------------------------------------
