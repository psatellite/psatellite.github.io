Path: Thermal/ThermalAnalysis

% Thermal equilibrium for a system with conduction and radiation
 This function computes the thermal equilibrium for a system with
 conductive and radiative heat exchange.
   
   lT^4 + kT = bq
   
 Contains a built-in demo which evalutates two nodes connected by a 
 conductor.

--------------------------------------------------------------------------
   Form:
   [t, err] = ThermalEquilibrium( d, q, t, nIts, epsilon )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   d           (:)   System matrices data structure
                     .k   linear term
                     .l   quadratic term
                     .b   input matrix
   q           (:)   Input fluxes
   t           (:)   Node temperatures
   nIts        (1,1) Number of iterations
   epsilon     (1,1) Error

   -------
   Outputs
   -------
   t           (:)   Node temperatures
   err         (:)   Node temperature error

--------------------------------------------------------------------------
