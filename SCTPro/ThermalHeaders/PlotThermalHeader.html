% Plots flux and temperature.
--------------------------------------------------------------------------
   Form:
   PlotThermal( q, temp, t, g, units )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   q              (:,:)  Flux. Each row is a component
   temp           (:,:)  Temperature. Each row is a component
   t              (1,:)  Time (sec)
   g              (1,1)  Spacecraft data structure
   units          {2}    {'Flux units' 'Temperature units' }

   -------
   Outputs
   -------
   None
