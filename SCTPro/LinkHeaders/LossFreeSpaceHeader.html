LossFreeSpace:

Path: Link/RF

% Computes the free space loss.

   This is the loss due to the spreading of the beam which goes
   as the distance squared.

   The default output is a 2D mesh plot.

   Type LossFreeSpace for a demo.

--------------------------------------------------------------------------
   Form:
   l = LossFreeSpace( f, r )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   f              (1,m)  Frequency (GHz)
   r              (1,n)  Distance (km)

   -------
   Outputs
   -------
   l              (n,m)  Loss (dB)

--------------------------------------------------------------------------
