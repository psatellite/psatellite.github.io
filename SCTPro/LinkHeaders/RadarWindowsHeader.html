RadarWindows:

Path: Link/Radar

% Parameters for different radar windows.
   'rectangle', 'triangle', 'cosine', 'hann', 'hamming', 'k-b 2.5'...
  'k-b 3.0', 'd-c 2.5', 'd-c 3.0'

   Parameters: Highest sidelobe level (dB)
               Sidelobe roll-off (dB/octave)
               Coherent gain
               Equivalent noise bandwdith (bins)
               3 dB Bandwidth (bins)
               Scallop loss (dB)
               Worst case process loss (dB);

   Form:
   y = RadarWindows( window )
--------------------------------------------------------------------------

   ------
   Inputs

   dBm           (1,:)    Window name

   -------
   Outputs
   -------
   y            (1,7)     Parameters
