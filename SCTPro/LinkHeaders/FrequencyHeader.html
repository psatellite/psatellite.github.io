Frequency:

Path: Link/LinkUtilities

% Outputs the frequencies for specified bands of electromagnetic radiation.
--------------------------------------------------------------------------
   Form:
       Frequency;
   f = Frequency( band, new, which )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   band      (1,:)  Band, for example 'uhf', 'visible'
   new       (1,1)  1 = Use new bands, i.e. 'a', 'b'
   which     (1,1) 'low' 'high' or 'both'

   -------
   Outputs
   -------
   f          (:)    Frequency (Hz)

--------------------------------------------------------------------------
