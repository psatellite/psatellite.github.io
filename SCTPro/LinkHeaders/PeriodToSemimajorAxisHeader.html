PeriodToSemimajorAxis:

Path: Link/LinkUtilities

% Computes semi-major axis from orbital period
 Computes semi-major axis from the period. This provides a convenient
 way to get repeat tracks by making the period an integral fraction of
 one day.

 Type PeriodToSemimajorAxis for a demo around the earth.
--------------------------------------------------------------------------
   Form:
   a = PeriodToSemimajorAxis( p, mu )
   a = PeriodToSemimajorAxis( p )     (Earth)
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   p   (1,:) Period (s)
   mu  (1,1) Gravitational parameter (km^3/s^2) [optional: default Earth]

   -------
   Outputs
   -------
   a   (1,:) Semi-major axis (km)  

--------------------------------------------------------------------------
 See also Period
