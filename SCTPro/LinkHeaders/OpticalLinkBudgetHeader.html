OpticalLinkBudget:

Path: Link/Optical

% Create an optical link budget.
   Assumes divergence limited transmitter.
   Has a built-in demo for  Pluto mission.
--------------------------------------------------------------------------
   Form:
   [pR, b] = OpticalLinkBudget( d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   d			  (:)   Input data structure
                       d.wavelength               Wavelength (m)
                       d.transmitApertureDiameter Transmit aperture (m^2)
                       d.thetaOffset              Pointing error (rad)
                       d.transmitPower            Transmission Power (W)
                       d.waveFrontError           Wavefront error
                                                  Loss is exp(-2*pi*error/wavelength)
                       d.range                    Range (m)
                       d.receiverLoss             Receiver loss (0 to 1)
                       d.receiverPointingLoss     Receiver pointing loss (0 to 1)
                       d.receiveApertureDiameter  Receiver aperture diameter (m)
                        
   -------
   Outputs
   -------
   pR            (1)     Receive power
   b             (:)     Budget struct

--------------------------------------------------------------------------
	References:	Lambert, S. G. and W. L. Casey (1995.) Laser Communications in
               Space. Artech, Boston.
--------------------------------------------------------------------------
