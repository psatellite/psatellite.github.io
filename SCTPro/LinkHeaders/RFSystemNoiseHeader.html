RFSystemNoise:

Path: Link/Radar

% Computes system noise.
   If tT is [] it is assumed to be 290 deg-K.
--------------------------------------------------------------------------
   Form:
   tS = RFSystemNoise( tA, tT, lR, f )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   tA            (1,1)    Antenna noise temperature (deg-K)   
   tT            (1,1)    Thermodynamic temperature (deg-K)     
   lR            (1,1)    Receiver ohmic losses (0 to 1)      
   f             (1,1)    Receiver noise figure (1+)    

   -------
   Outputs
   -------
   tS            (1,1)    System noise temperature (deg-K)     

--------------------------------------------------------------------------
