GainPattern:

Path: Link/RF

% Compute an antenna gain pattern as a function of the incidence angle.

 Compute an antenna gain pattern as a function of the incidence angle for
 a circular dish antenna.

 Type GainPattern for a demo.

--------------------------------------------------------------------------
   Form:
   g = GainPattern( antDiam, centerFreq, angle );
--------------------------------------------------------------------------

   -------
   Inputs:
   -------
   antDiam       (1,1)   Antenna diameter  (m)
   centerFreq    (1,1)   Center frequency (MHz)
   angle         (1,:)   Incidence angle (rad)

   --------
   Outputs:
   --------  
   g             (1,:)   Gain 

--------------------------------------------------------------------------
