CNRComp:

Path: Link/LinkUtilities

% Compute the carrier to noise ratio.
 The inputs are the probability of false alarms, probability of detection and 
 number of pulses.

 Run a demo by typing CNRComp. See also FPD.
--------------------------------------------------------------------------
   Form:
   cNR = CNRComp( pFA, pD, n )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   pFA           (1,1)  Probability of false alarm
   pD            (1,1)  Probability of detection
   n             (1,:)  Number of pulses

   -------
   Outputs
   -------
   cNR           (1,:)   Carrier to noise ratio

--------------------------------------------------------------------------
