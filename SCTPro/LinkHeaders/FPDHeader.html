FPD:

Path: Link/LinkUtilities

% Function called by fzero to compute the carrier to noise ratio. 
--------------------------------------------------------------------------
   Form:
   x = FPD( cNR, pD, yB, n )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   CNR           (1,1)  Carrier to noise ratio
   pD            (1,1)  Probability of detection
   yB            (1,1)  Upper range of integrand
   n             (1,:)  Number of pulses

   -------
   Outputs
   -------
   x             (1,1)   Value

   See also CNRComp.
--------------------------------------------------------------------------
