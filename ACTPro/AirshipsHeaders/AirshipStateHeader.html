AirshipState:

Path: Airships/Control

%vCompute the airship state vector as an acstate object from a model data
   structure and initial conditions of:
     [latitude, longitude, altitude]
     [airspeed, angle of attack, sideslip]
     [roll, pitch, yaw]

--------------------------------------------------------------------------
   Form:
   x = AirshipState( d, latLonAlt, vAB, eul );
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   d              (.)    Initial aircraft data structure, generated from
                          "BuildAirshipModel", or you may supply just the
                          name of an airship model.
   latLonAlt     (4,1)   Latitude (deg), Longitude (deg), Altitude (km),
                          4th element is Julian Date (default is JD2000)
   vAB           (3,1)   Velocity (m/s), alpha (rad), beta (rad)
   eul           (3,1)   Roll, Pitch, Yaw (rad)

   -------
   Outputs
   -------
   x              (.)    Initial aircraft state, class "acstate"

--------------------------------------------------------------------------
