Introduction

This GUI allows you to plot data generated by the telemetry display.
You first push the Load Data button to load data into the GUI. If you have
saved a format you can push Load Format and load the saved format. To
generate a plot:
1. Load Data.
2. (Optional) Load Format.
3. Enter a page name and push New Page.
4. Push the plot buttons to form a plot pattern. You can go from
   1-by-1 to 4-by-2.
5. Click Edit
6. Enter a Plot Title
7. Select parameters to plot from the plot list.
8. Push Store Plot
9. Repeat for the remaining Plot buttons.
10. Push Plot to plot the data
11. Add new pages as  needed.

Buttons

  • Load Data: Load data generated by the Telemetry GUI
  • Load Format: Load a format.
  • Close plots: This edit box gives the number of points to be saved to a file.
  • Format: Opens a window that allows you to set the display format for a telemetry point. This format will apply to all pages. Use standard C formats:
  • New Page: Add a new page. The page will appear in the right-hand list.
  • Delete Page: Delete the page selected in the list.
  • Select EPSF Dir: Select a directory to store EPSF files.
  • Save Plots to EPSF: Generate encapsulated Postscript files for each page.
  • Add a Page:
  • Save Plot Format: Save the plot format in a file.
  • PLot: Plot the data.
  • Quit: Quit the Telemetry GUI.
  • Help: Opens the help window
