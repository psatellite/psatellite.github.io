Introduction

This help file explains how to use the AnimationGUI. The AnimationGUI helps you visualize and simulate objects in 3D. It allows you to play the animation forward and backward as well as pause it. You can zoom in and out as well as change the sampling rate to slow down or speed up the animation speed.

Getting Started

To run a demo animation, simply type AnimationGUI. This will allow you to get a feel for all the controls it has to offer.

Buttons

< (play backwards)

Play the animation backwards.

|< (step back)

Move one frame backwards. Can only be used while paused.

(pause)

Pause the animation

>| (step)

Move one frame forwards.

> (play)

Play the animation forwards.

O (Record)

Records the animation

Reset

Resets the animation to the initial position.

+ (Zoom)

Zoom in on the animation

- (Zoom)

Zoom out of the animation

Show Name

Shows the names of the objects in the animation next to the corresponding object.

Sliders

Sample Rate

Changes the sample rate for state data (and therefore the speed of the animation).

Tabboxes

Equal

This toggles the axis increments to be proportional or not (will distort animation).

Tail

Toggles a trailing tail to show the recent path of the objects

Path

Toggles a path wire from start to finish of the path the object takes.
