Overview of New Features

  • Several new demos
  • Aircraft performance functions for takeoff, landing, gliding, climb and turns
  • New utilities in Common module
  • Aircraft fuel consumption functions
  • New drag functions with drag data for several aircraft
  • Improved group mass calculation functions
