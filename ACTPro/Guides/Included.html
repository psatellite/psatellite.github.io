What's Included

The Aircraft Control Toolbox (ACT) is a library of MATLAB m-files and mat-files. The toolbox is organized by subject into a few different top-level modules, and each module is further organized into several different sub-folders.

ACT provides you with a variety of tools for the modeling, simulation, and control system design of aircraft. There is a substantial set of software which the Aircraft Control Toolbox shares with the Spacecraft Control Toolbox, and this software is in the modules AeroUtils, Common, Math, and Plotting. The aircraft files are in AC which also includes special CAD models. Professional editions also include the ACPro and Airships modules.

Toolbox Modules

AC Coordinate transformations, aircraft models, integrated simulation, standard atmosphere, aerodynamic property calculations, basic control designs.
AeroUtils Atmospheric models, CAD functions and components, coordinate transformations.
Common Quaternions, control, estimation, time conversions, graphics and general utilities.
Math Math utilities including geometry, integration, probability, and solvers.
Plotting Plotting tools for complex simulations including animation.

Professional Edition

ACPro Engine models, flexible dynamics model, more aircraft models, performance analysis tools, point mass trajectory simulation, wind disturbance models.
Airhips Modeling, simulation, and control of airships.
