Overview of New Features

We added new functions for aircraft trajectory analysis. You can now script an aircraft to fly a trajectory including ground roll, takeoff, climb and flight at altitude. Each segment can included acceleration and deceleration. New easier-to-use point mass models and control systems were added.

New design tools were added for computing mass budgets and estimating costs. new modeling tools were added along with many "helper" functions to make it easiers to design, analyzed and simulated aircraft.
