% Log base b
 Either x or b can be a vector. 
------------------------------------------------------------------------
   Form:
   y = logB(b,x)
------------------------------------------------------------------------

   ------
   Inputs
   ------
   b 	(1,:)	Base
   x   (1,:) Value

   -------
   Outputs
   -------
   y 	(1,:)	log(x) base b
