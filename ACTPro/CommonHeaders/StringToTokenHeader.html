StringToToken:

Path: Common/General

% Finds the first token in a string

--------------------------------------------------------------------------
   Form:
   [t, r] = StringToToken( s )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   s       (:)   String (may be a cell element)

   -------
   Outputs
   -------
   t       (1,:) Character string with the token
   r       (1,:) Remainder of the string

--------------------------------------------------------------------------
