SetUIElement:

Path: Common/General

% Set certain properties of UIElements 

 property  
 --------
 'visible' 'on', 'off'
 'enable'  'on', 'off'
 'position' [x y width height]

 If the style is not for a UIElement, it will automatically pass the 
 arguments to uicontrol.

--------------------------------------------------------------------------
   Form:
   SetUIElement( h, varargin )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   h          (:)   Handle
   varargin   {:}   Property pairs
       
   -------
   Outputs
   -------
   none

--------------------------------------------------------------------------
