KronProd:

Path: Common/Control

% Compute the Kronecker product.

--------------------------------------------------------------------------
   Form:
   c = KronProd( a, b )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   a             (m,n)  Matrix
   b             (p,q)  Matrix

   -------
   Outputs
   -------
   c             (mp,nq) Kronecker product

--------------------------------------------------------------------------
   References:	Zhou, K., et. a. Robust and Optimal Control, Prenctice-Hall,
               1996, p. 25.
--------------------------------------------------------------------------
