FSWClock:

Path: Common/General

% This routine implements the flight software clock.
--------------------------------------------------------------------------
   Form:
   t = FSWClock( action, d );
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   action        (1,:)     Action to be performed        
   d             (1,1)     Data structure for inputs

   -------
   Outputs
   -------
   t             (1,1)     Data structure with outputs. Valid only
                           with action 'update' and 'get telemetry'

--------------------------------------------------------------------------

Common: Time/JD2000
