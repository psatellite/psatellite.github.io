DateStringToJD:

Path: Common/Time

% Convert a date string to Julian date. The date string
 must be in the form 4/20/2000 00:00:00 or 4/20/2000

 Type DateStringToJD for a demo
--------------------------------------------------------------------------
   Form:
   jD = DateStringToJD( s )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   s             (1,:)   String 'mm/dd/yyyy hh:mm:ss' or 'mm/dd/yyyy'
                         or 'mm/dd/yyyy hh:mm'

   -------
   Outputs
   -------
   jD            (1,1)   Julian Date   

--------------------------------------------------------------------------
