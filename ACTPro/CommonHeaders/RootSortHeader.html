RootSort:

Path: Common/Control

% Sort columns of roots for Root Locus. 

   Each column is a root list generated by eig. It tries to match the
   roots in column k with those in column k-1.
--------------------------------------------------------------------------
   Form:
   s = RootSort( r )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   r              (n,m)  Roots

   -------
   Outputs
   -------
   s              (n,m)  Sorted roots
