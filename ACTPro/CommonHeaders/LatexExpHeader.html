LatexExp:

Path: Common/General

% Creates a latex exponential string

 Converts a number like 1.2e3 in 1.2$\times 10^{3}$.

 Type LatexExp for a demo.

 See also LatexScientificNotation.
------------------------------------------------------------------------
   Form:
   s = LatexExp( n, f )
------------------------------------------------------------------------

   ------
   Inputs
   ------
   n     (1,1) Number
   f     (1,:)	Exponential format (don't include the e)

   -------
   Outputs
   -------
   s     (1,:)	Latex string
