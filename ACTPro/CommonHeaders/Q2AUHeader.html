Q2AU:

Path: Common/Quaternion

% Convert a quaternion to an angle and a unit vector. 
 If q = [1;0;0;0] then u is set to [1;0;0].

--------------------------------------------------------------------------
   Form:
   [angle, u] = Q2AU( q )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   q       (4,1)   Quaternion

   -------
   Outputs
   -------
   angle   (1,1)   Angle
   u       (3,1)   Unit vector

--------------------------------------------------------------------------
