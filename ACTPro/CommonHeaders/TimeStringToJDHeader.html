TimeStringToJD:

Path: Common/Time

% Convert a time string to Julian date. The time string
 must be in the form 00:00:00.

 Type TimeStringToJD for a demo.
--------------------------------------------------------------------------
   Form:
   jD = TimeStringToJD( s )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   s             (1,:)   String 'hh:mm:dd'

   -------
   Outputs
   -------
   jD            (1,1)   Julian Date   

--------------------------------------------------------------------------

Children:

Common: Time/DTSToDTA
Common: Time/Date2JD
