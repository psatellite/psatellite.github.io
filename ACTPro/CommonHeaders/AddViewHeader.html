AddView:

Path: Common/Graphics

% Add view accelerators to a figure. 
 Snap to different frames as shown:

   x-y   CTRL+J
   y-x   CTRL+K
   x-z   CTRL+L
   y-z	  CTRL+M
--------------------------------------------------------------------------
   Form:
   AddView( h )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   h                (:)  Handle to the figure (optional, default is gcf)

   -------
   Outputs
   -------
   none
