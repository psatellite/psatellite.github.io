MatrixCellTo3D:

Path: Common/General

% Convert a cell array of p x p matrices into a 3D matrix. 
--------------------------------------------------------------------------
   Form:
   m = MatrixCellTo3D( c )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   c             {n}       Cell array of m x m matrices

   -------
   Outputs
   -------
   m             (p,p,n)   3D matrix
