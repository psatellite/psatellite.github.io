% Transform a a 3x3 inertia matrix into 6x1 compact inertia vector.

   Vector form:   [Ixx; Iyy; Izz; Ixy; Ixz; Iyz]

--------------------------------------------------------------------------
   Form:
   IVec = IConvP( IMat );
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   IMat          (3,3)    Full inertia matrix

   -------
   Outputs
   -------
   IVec          (6,1)    Compact inertia vector
