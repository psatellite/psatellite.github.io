Path: Common/General

% Converts an integer to binary [lsb msb] in two's complement.
--------------------------------------------------------------------------
   Form:
   b = IntToBinary( i )
--------------------------------------------------------------------------

   ------
   Inputs
   ------

   i            (1,1) Integer

   -------
   Outputs
   -------
   b            (1,:) Binary array [lsb, msb]
