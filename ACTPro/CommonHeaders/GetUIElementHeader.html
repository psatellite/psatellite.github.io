GetUIElement:

Path: Common/General

% Get certain properties of UIElements.

 property   
 --------
 'visible'
 'position'
 'datastructure'

 If the handle is for a uicontrol it will return
 the property of that uicontrol.

--------------------------------------------------------------------------
   Form:
   x = GetUIElement( h, property )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   h          (:)    Handle
   property   (1,:)  Property
       
   -------
   Outputs
   -------
   x          {:}    Depends on the property

--------------------------------------------------------------------------
