Path: Common/Control

% Propagates state equations.

--------------------------------------------------------------------------
   Form:
   [y, x] = StatePrp( a, b, c, d, x, u, sType ) 
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   a                      State transition matrix
   b                      Input matrix
   c                      Measurement matrix
   d                      Feedthrough matrix
   x                      State
   u                      Input
   sType                  If omitted z-transform, else delta.

   -------
   Outputs
   -------
   y                      Output
   x                      State
