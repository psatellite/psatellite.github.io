DispWithTitle:

Path: Common/General

% Display a variable with a title.

 x can be any MATLAB variable. If the title is left out, the variable name
 will be displayed via inputname. The variable is displayed using disp.

 Type DispWithTitle for a demo.
--------------------------------------------------------------------------
   Forms:
	  DispWithTitle( x, t )
	  DispWithTitle( x )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x   :     Variable
   t   (1,:) Title

   -------
   Outputs
   -------
   None

--------------------------------------------------------------------------
