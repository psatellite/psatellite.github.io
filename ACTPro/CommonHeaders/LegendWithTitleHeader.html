% Create a legend with a title.
 legend uses 'tex' as the interpreter so you can pass it latex
 commands in s or in title.
--------------------------------------------------------------------------
   Form:
   LegendWithTitle( s, title, varargin )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   s         {:}   Legend entries
   title     (1,:) Title
   vararagin {:}   Parameter pairs passed to legend

   -------
   Outputs
   -------
   None
