Truss:

Path: Common/ComponentModels

% Generate a square truss going from x1 to x2.
 If n is omitted n = 10. wBar is optional and is the width of the struts.
 Note that 2D trusses will not work properly if exported as OBJ files.

 Type Truss for a demo of a truss with 3D struts.
--------------------------------------------------------------------------
   Form:
   [v, f, l] = Truss( x1, x2, w, n, wBar )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x1         (3,1) Location of top
   x2         (3,1) Location of bottom
   w          (1,1) Width
   n          (1,1) Number of divisions
   wBar       (1,1) Bar width

   -------
   Outputs
   -------
   v         (:,3)  Vertices
   f         (:,3)  Faces
   l         (1,1)  Total length of the truss elements

--------------------------------------------------------------------------
