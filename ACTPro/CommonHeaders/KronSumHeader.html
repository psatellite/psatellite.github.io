KronSum:

Path: Common/Control

% Compute the Kronecker sum.

--------------------------------------------------------------------------
   Form:
   c = KronSum( a, b )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   a             (n,n)  Matrix
   b             (m,m)  Matrix

   -------
   Outputs
   -------
   c             (nm,nm) Kronecter Sum

--------------------------------------------------------------------------
   References:	  Zhou, K., et. a. Robust and Optimal Control, Prenctice-Hall
                 1996, p. 25.
--------------------------------------------------------------------------
