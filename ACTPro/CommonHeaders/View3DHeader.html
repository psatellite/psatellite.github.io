View3D:

Path: Common/Graphics

% View a plane of a 3D plot or standard cartesian view.
 The default view is view(3).

 View3D is the same as View3D('none')
--------------------------------------------------------------------------
   Forms:
   View3D( 'xy' )
   View3D( 'yz' )
   View3D( 'xz' )
   View3D( 'cart' )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   type     (:)    String indicating view type.

   -------
   Outputs
   -------
   none

--------------------------------------------------------------------------
