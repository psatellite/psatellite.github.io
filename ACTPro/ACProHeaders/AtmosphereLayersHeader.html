AtmosphereLayers:

Path: ACPro/Atmosphere

% Names the atmospheric layer for altitude h. 

   Type AtmosphereLayers for a picture of the atmosphere.
--------------------------------------------------------------------------
   Form:
   layer = AtmosphereLayers( h )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   h              (1,1)  Altitude (km)

   -------
   Outputs
   -------
   layer          (1,:)  Name of layer

--------------------------------------------------------------------------
   Reference: http://www.ace.mmu.ac.uk/eae/Atmosphere/Older
            /Atmospheric_Layers.html
