Scimitar:

Path: ACPro/Airbreathing

% Models a Scimitar engine from Reaction Engines, Ltd. 
   Uses lookup tables from publicly available data. It interpolates
   from the input data. This model doesn't use d or q. The model assumes
   a particular Mach number altitude profile.

   Type Scimitar for a demo.

--------------------------------------------------------------------------
   Form:
   [t, sFC, h] = Scimitar( m, d, q )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   m           (1,1)  Mach number
   d           (1,1)  Engine data structure (not used)
   q           (1,1)  Standard atmosphere (not used)

   -------
   Outputs
   -------
   t           (1,1)  Thrust (N)
   sFC         (1,1)  Specific Fuel Consumption (kg/N-s)
   h           (1,1)  Assumed altitude (m)

--------------------------------------------------------------------------
   Reference: https://spiral.imperial.ac.uk:8443/bitstream/10044
              /1/1456/1/EUCASS07_scimitar_5_08_03.pdf
--------------------------------------------------------------------------
