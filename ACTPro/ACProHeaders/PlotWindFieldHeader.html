PlotWindField:

Path: ACPro/Wind

% Plot the wind field from a data structure array. 
 Each element of the  has a single lat/lon coordinate, and a vector of 
 wind speeds and headings across an altitude range. 
 
 See also: NOAAWindData, MergeMOAAWindData
--------------------------------------------------------------------------
   Form:
   PlotWindField( windData )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   d           (.)     Wind data set from NOAAWindData.
   altRange   (1,2)    Altitude range. Optional. (km)
   fig         (1)     Figure handle. Optional.

   -------
   Outputs
   -------
   fig         (1)     Figure handle.

--------------------------------------------------------------------------

Common: Graphics/NewFig
