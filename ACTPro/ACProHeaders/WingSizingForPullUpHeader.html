WingSizingForPullUp:

Path: ACPro/ACPerformance

% Sizes a wing for dynamic pull up.

   Type WingSizingForPullUp for a demo
--------------------------------------------------------------------------
   Form:
   s = WingSizingForPullUp( pullUp, m, h, cLAlpha, alpha, thrust, mass, sweep )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   pullUp        (1,1)    Pull up (g)
   m             (1,1)    Mach number
   cLAlpha       (1,1)    Lift coefficient for alpha (1/rad)
   thrust        (1,1)    Engine thrust (N)
   alpha         (1,1)    Angle of attack (rad)
   mass          (1,1)    Mass (kg)
   sweep         (1,1)    Sweep angle (rad)

   -------
   Outputs
   -------
   s             (1,1)    Wing area (m^2)

--------------------------------------------------------------------------
