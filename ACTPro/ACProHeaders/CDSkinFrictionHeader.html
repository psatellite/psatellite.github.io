CDSkinFriction:

Path: ACPro/ACProAerodynamics

% Skin friction coefficient using a variety of models.

   The default is Prandtl-Schlichting. The default standard atmosphere
   is at 1000 m. 'miller' or 'compressible' should be used for
   supersonic flow.

   It uses 5e5 for Reynold's number at low speeds so that there isn't a
   low-speed singularity.

   Type CDSkinFriction for a demo of Prandtl-Schlichting and the
   compressible models.

--------------------------------------------------------------------------
   Form:
   cF = CDSkinFriction( m, l, p, model )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   m          (1,:) Mach number
   l          (1,1) Characteristic length (m)
   p           (.)  Standard atmosphere data structures
   model      (1,:) Model. Available models:
                          'prandtl-schlichting'
                          'prandtl'
                          'kempf-karman'
                          'schultz-grunov'
                          'miller'
                          'compressible'

   -------
   Outputs
   -------
   cF         (1,:) Skin friction

--------------------------------------------------------------------------
   References: http://www.cfd-online.com/Wiki/Skin_friction_coefficient
--------------------------------------------------------------------------
