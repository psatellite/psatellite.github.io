ParasiticDragCoefficient:

Path: ACPro/ACProAerodynamics

% Parasitic drag coefficient for a set of aircraft components.
--------------------------------------------------------------------------
   Form:
   cP = ParasiticDragCoefficient( d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   d          (:)   Array of data
                    .u      (1,:) Velocity (m/s)
                    .h      (1,:) Altitude (m)
                    .component
                         .type (1,:) 'fuselage', 'wing', 'nacelle'
                         .l    (1,1) Length (m)
                         .d    (1,1) Diameter (m)
                         .s    (1,1) Surface area (m)
                         .q    (1,1) Interference factor 0-1
                         .r    (1,1) t/c for wings, L/D for all others

   -------
   Outputs
   -------
   cP         (1,:)  Parasitic drag coefficient

--------------------------------------------------------------------------
   See also WettedArea, AeroFormFactor, SkinFriction
--------------------------------------------------------------------------
