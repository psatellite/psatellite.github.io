Path: ACPro/Trajectory

% Determines the set points to achieve a trajectory
--------------------------------------------------------------------------
   Form:
   [segment, setPoint, distance, time] = Trajectory( x, t, segment, 
                                           setPoint, distance, dT, time )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x         (6,1) State
   t         (1,1) Time (s)
   segment   (1,1) Segment number
   setPoint  (.)   Data structure 
                   .v        	(1,1) Velocity (m/s)
                   .h         	(1,1) Altitude (m)
                   .d         	(1,1) Distance (m)
                   .psi        (1,1) Heading (rad)
                   .transition	{:}   Transition type
                   .phi        (1,1) Bank angle (rad)
                   .gamma      (1,1) Flight path angle (rad)
   distance  (1,1) Distance since start of segment
   time      (1,1) Time since start of segment

   -------
   Outputs
   -------
   segment   (1,1) Segment number
   setPoint  (.)   Data structure 
   distance  (1,1) Distance since start of segment
   time      (1,1) Time since start of segment

--------------------------------------------------------------------------
