% Look up Boeing 737-800 data parameters
 Calling without input produces a list of available parameters. All results are
 mks.
--------------------------------------------------------------------------
   Form:
   v = Boeing737(arg)
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   arg (1,:) Parameter, 'aspect ratio, 'wing area', etc.
                        
   -------
   Outputs
   -------
   v   (1,1) Value of request parameter
                        
--------------------------------------------------------------------------

               Junzi Sun, Jacco M. Hoekstra, Joost Ellerbroek,"Aircraft 
               Drag Polar Estimation Based on a Stochastic Hierarchical
               Model."
