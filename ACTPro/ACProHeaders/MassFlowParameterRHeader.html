MassFlowParameterR:

Path: ACPro/ACEngine

% Mass flow parameter with gamma, r and Mach number as inputs. 
  An alternative function is MassFlowParameter which takes cP
  as an input.
--------------------------------------------------------------------------
   Form:
   mFP = MassFlowParameterR( gamma, r, machNo, units )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   gamma      (1,1)   Rate of specific heats
   r          (1,1)   Gas constant
   machNo     (1,:)   Mach number
   units      (1,:)   'eng' or 'mks'

   -------
   Outputs
   -------
   mFP        (1,:)   Mass flow parameter

--------------------------------------------------------------------------
