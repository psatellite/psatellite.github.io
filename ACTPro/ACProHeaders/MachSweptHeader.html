% Effective Mach number with sweep.

   Type MachSwept for a demo.
--------------------------------------------------------------------------
   Form:
   mEff = MachSwept( m, sweep )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   m        (1,:)  Mach number
   sweep	   (1,:)  Sweep (deg)

   -------
   Outputs
   -------
   mEff     (1,:)  Effective Mach number

--------------------------------------------------------------------------
   Reference: Anderson, J. D., "Introduction to Flight, 3rd Edition", 
              McGraw-Hill, 1989, pp. 227-228
