GeoMagLatLon:

Path: ACPro/Atmosphere

% Geomagnetic latitude and longitude.

   Type GeoMagnLatLon for a demo.
--------------------------------------------------------------------------
   Form:
   [lat, lon] = GeoMagLatLon( r )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   r              (3,:)  Position vector

   -------
   Outputs
   -------
   lat            (1,:)  Latitude (rad)
   lon            (1,:)  Longitude (rad);

--------------------------------------------------------------------------
   Reference: http://www.cosmicrays.org/muon-cutoff-rigidity.php
--------------------------------------------------------------------------
