InletAndNozzle:

Path: ACPro/CAD

% Generate vertices for an inlet and a nozzle.
 The inlet is +x.

 Type InletAndNozzle for a demo.
--------------------------------------------------------------------------
   Form:
   [v, f] = InletAndNozzle( l, w, h, angle )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   l     (1,1)  x length
   w     (1,1)  y width
   h     (1,1)  z height
   angle	(1,1)  Inlet ramp angle
   

   -------
   Outputs
   -------
   v      (:,3) Vertices
   f      (:,3) Faces

--------------------------------------------------------------------------
