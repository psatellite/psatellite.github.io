CreateMVSSetupFileEntries:

Path: ACPro/AircraftPro

--------------------------------------------------------------------------
   Creates setup file entries for MultiVehicleSim
--------------------------------------------------------------------------
   Form:
   CreateMVSSetupFileEntries( d, r, vBody, w, cG, eulerAngle, engine, scale )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   d              (1,1)  
                         .control
                                  .rudder   (1,1) Rudder value (deg)
                                  .aileron  (1,1) Differential aileron value (deg)
                                  .elevator (1,1) Elevator value (deg)
                                  .throttle (1,1) Throttle value
   r              (3,1)  EF position vector (ft)
   vBody          (3,1)  Body frame velocity vector (ft/sec)
   w              (3,1)  Angular rate vector (rad/sec)
   cG             (3,1)  CG vector (fractional)
   eulerAngle     (3,1)  [roll;pitch;yaw] (rad)
   engine         (1,1)  Engine state
   scale          (1,1)  Control input scale (units to counts)

   -------
   Outputs
   -------
   None

--------------------------------------------------------------------------
