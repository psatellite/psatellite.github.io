AirlinerSeats:

Path: ACPro/ACDesign

% Layout airliner seats.

 Allows for first class, business class and economy.

 Type AirlinerSeats for a demo.

--------------------------------------------------------------------------
   Form:
   d = AirlinerSeats( d, p )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   d	(1,:)  Data structure
              .type        (1,:) 'First', 'Business', 'Economy', 'Other'
              .nRows       (1,1) Number of rows
              .pRow        (1,:) Pattern per row
   p   (1,1)  Data structure with parameters
              .pitch       (1,3) [first business economy]
              .width       (1,3) [first business economy] 
              .widthAisle	(1,3) [first business economy] 
              .widthExit   (1,1) Exit aisle 
              .exits       (1,1) Exit location row 
   -------
   Outputs
   -------
   d	(1,1)  Data structure

--------------------------------------------------------------------------
