F18Model:

Path: ACPro/AircraftPro

% Generates a state space F-18 model from a range of models. 
   The mach #s range from .35 to 0.7 and altitudes from 15 kFt to 32 kFt.
--------------------------------------------------------------------------
   Form:
   [g, machNo, altitude, def] = F18Model( machNo, altitude )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   machNo         (1,1)  Mach number
   altitude       (1,1)  Altitude (kFt)

   -------
   Outputs
   -------
   g              (1,1)  State space model
   machNo         (1,1)  Nearest Mach number
   altitude       (1,1)  Nearest altitude (kFt)
   def            {:,3}  State, input and output definitions

--------------------------------------------------------------------------
