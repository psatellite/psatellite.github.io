AutoCADColor:

Path: AerospaceUtils/CAD

% Converts an AutoCAD color into a [r,g,b] color.

 Type AutoCADColor for demo.
--------------------------------------------------------------------------
   Form:
   rGB = AutoCADColor( k )
--------------------------------------------------------------------------

   ------
   Inputs
   ------

   k            (1,:) AutoCAD color from 0 to 255

   -------
   Outputs
   -------
   rGB          (1,3) [r, g, b]

--------------------------------------------------------------------------
   Reference: Rule, K. (1996.) 3D Graphics File Formats. Addison-Wesley.
--------------------------------------------------------------------------
