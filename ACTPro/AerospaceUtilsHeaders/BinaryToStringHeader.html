BinaryToString:

Path: AerospaceUtils/CAD

% Converts a bit array to a string. [1 0 1 0] will become 0101.

 Type BinaryToString for demo.
--------------------------------------------------------------------------
   Form:
   s = BinaryToString( b )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   b            (1,:) Binary array

   -------
   Outputs
   -------
   s            (1,:) String
