InterpolateCircle:

Path: Math/Analysis

% Interpolates between x1 and x2 when x1 and x2 lie in a circle. When
 f = 0 x = x1, when f = 1 x = x2.
--------------------------------------------------------------------------
   Form:
   x = InterpolateCircle( x1, x2, f )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x1            (:,1)    First quantity
   x2            (:,1)    Second quantity
   f             (1,:)    0 to 1

   -------
   Outputs
   -------
   x             (:,1)    Solution

--------------------------------------------------------------------------
