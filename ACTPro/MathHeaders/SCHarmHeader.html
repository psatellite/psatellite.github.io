SCHarm:

Path: Math/Analysis

% Generate a series of sine and cosine harmonics of the arguments

--------------------------------------------------------------------------
   Form:
   [s, c] = SCHarm( a, n )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   a                  column vector (m,1) argument (rad)
   n                  number of harmonics

   -------
   Outputs
   -------
   s                  vector of sine harmonics
   c                  vector of cosine harmonics

--------------------------------------------------------------------------
