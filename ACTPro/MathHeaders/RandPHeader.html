RandP:

Path: Math/Probability

% Generate a Poission random matrix.
 Type RandP for a demo.
--------------------------------------------------------------------------
   Form:
   q = RandP( lambda, n, m )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   lambda    (1,1) Coefficient
   n         (1,1) Rows
   m         (1,1) Columns

   -------
   Outputs
   -------
   q         (n,m) Random matrix

--------------------------------------------------------------------------
   Reference: Knuth
--------------------------------------------------------------------------
