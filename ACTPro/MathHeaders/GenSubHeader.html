Path: Math/Linear

% Subtract two matrices where NaN - NaN or inf - inf == 0

   Since version 2.
--------------------------------------------------------------------------
   Form:
   c = GenSub( a, b )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   a             (n,n)  Matrix
   b             (n,n)  Matrix

   -------
   Outputs
   -------
   c             (n,n) Difference
