SkewSq:

Path: Math/Linear

% Computes the product of two skew symmetric matrices derive from vectors.
   If only one matrix is entered it will compute the square.

   Since version 2.
--------------------------------------------------------------------------
   Form:
   s = SkewSq( v, u )
--------------------------------------------------------------------------

   -----
   Input
   -----
   v               (3,1) Vector
   u               (3,1) Vector

   ------
   Output
   ------
   s               (3,3) Square of the skew matrix
