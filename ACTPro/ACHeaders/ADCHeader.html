Path: AC/Aero

% Implements the "Air Data Computer" model.
--------------------------------------------------------------------------
   Form:
   [aMach, qBar] = ADC( vT, alt )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   vT             (1,1)  Velocity (ft/sec)
   alt            (1,1)  Altitude (ft)

   -------
   Outputs
   -------
   aMach          (1,1)  Mach Number
   qBar           (1,1)  Dynamic pressure
