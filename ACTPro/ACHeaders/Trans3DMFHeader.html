Trans3DMF:

Path: AC/ACGraphics

% Transform a 3D data structure. r is optional.
--------------------------------------------------------------------------
   Form:
   g = Trans3DMF( m, g, r )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   m           (3,3) Transformation matrix.
   g           (.)   Data structure
                     .m    (:)   Data structure for meshes
                                 .v                  (:,3) Vertices
                                 .f                  (:,:) Faces
                                 .ambientCoefficient (1,1) 
                                 .diffuseColor       (3,1)
                                 .specularColor      (3,1)
                                 .specularControl    (1,1)
                                 .transparencyColor  (3,1)
                     .name (1,:) Name
   r           (3,1) Position vector

   -------
   Outputs
   -------
   g           (.) Transformed data structure

--------------------------------------------------------------------------

Children:

Math: Linear/DupVect
