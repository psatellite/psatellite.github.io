AeroD:

Path: AC/Aircraft

% Compute the state space system based on aerodynamic derivatives.
   See DC8.m for an example of the database format.
   The state vector is either:

   'wind':      [vT; alpha; beta; p; q; r; phi; theta; psi]

   or

   'stability': [u; v; w; p; q; r; phi; theta; psi]

--------------------------------------------------------------------------
   Form:
   [g, gLat, gLon] = AeroD( f, axisSystem )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   f                     Aircraft data or database for input
   axisSystem            'wind', 'stability'

   -------
   Outputs
   -------
   g                     State space system of type statespace
   gLat                  Lateral dynamics
   gLon                  Longitudinal dynamics

--------------------------------------------------------------------------
