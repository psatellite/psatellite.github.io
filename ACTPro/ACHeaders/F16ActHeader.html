F16Act:

Path: AC/ACDemoFuns

% Models the F16 actuators as first order lags with time constants
   based int by actuatorData
--------------------------------------------------------------------------
   Form:
   [dX, control] = F16Act( x, controlInput, actuatorData )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x             (:)   State
   controlInput  (:)   Control commands
   actuatorData  (:)   Data

   -------
   Outputs
   -------
   dX          (:)   State derivatives
   control     (:)   Controls
