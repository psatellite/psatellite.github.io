DrawTerrain:

Path: AC/ACGraphics

% Draw terrain.

--------------------------------------------------------------------------
   Form:
   [h, v] = DrawTerrain( type, d, hAxes )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   type        (1,:) Type 'flat' is the only option
   d           (1,1) Dimension data, for 'flat'
                     d.w width of grid
                     d.n number of points
   hAxes       (1,1) Handle to axes

   -------
   Outputs
   -------
   h                 Handle to the patch
   v                 Vertices

--------------------------------------------------------------------------
