DTAToDTS:

Path: Common/Time

% Converts the date time array to the date time structure

 Typing DSTToDTA returns the clock time.

--------------------------------------------------------------------------
   Form:
   dateTime = DTSToDTA( t )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   t           (1,6)   [year month day hour minute seconds 

   -------
   Outputs
   -------
   dateTime    (1,1)   Date/time structure
                       .year
                       .month                             
                       .day                             
                       .hour                             
                       .minute                             
                       .second

--------------------------------------------------------------------------
