ColumnSizeCellArray:

Path: Common/General

% Outputs the size of each column of a cell array.
 Elements of m can be strings or scalars.

--------------------------------------------------------------------------
   Form:
   s = ColumnSizeCellArray( m, f )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   m               (n,m)  Cell array
   f               (1,1)  Width of number fields

   -------
   Outputs
   -------
   f               (1,m)  Maximum width of each column

--------------------------------------------------------------------------
