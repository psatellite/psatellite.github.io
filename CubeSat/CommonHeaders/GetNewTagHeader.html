GetNewTag:

Path: Common/General

% Get a new tag to uniquely identify a figure. 
 Uses clock and rand to generate a unique string.
--------------------------------------------------------------------------
   Form:
   tag = GetNewTag( name )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   name        (1,:)  Name of the figure

   -------
   Outputs
   -------
   tag         (1,:)  Tag
