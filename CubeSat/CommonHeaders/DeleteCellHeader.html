DeleteCell:

Path: Common/General

% Delete a cell element of a one dimensional cell array.
 If k is a logical array, the cells that will be deleted are the ones
 with true values, i.e. c = c(~k)
--------------------------------------------------------------------------
   Form:
   c = DeleteCell( c, k )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   c        {}  Cell array
   k        (:) Array of elements, or logical array

   -------
   Outputs
   -------
   c        {}  Cell array

--------------------------------------------------------------------------
