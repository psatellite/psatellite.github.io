LabelLine:

Path: Common/Graphics

% Labels a line.
 x must have at least 2 rows. 
 The format string is any standard matlab format such as 't = %12.4f'

--------------------------------------------------------------------------
   Form:
   LabelLine( x, t, n, f )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x          (2,:)   [x;y]
   t          (1,:)   Numbers for labels
   n          (1,1)   Number of labels
   f          (1,:)   Format string

   -------
   Outputs
   -------
   None

--------------------------------------------------------------------------
