% Magnitude of a set of vectors
 Given a 3-by-n matrix where each column represents a vector, return a row
 vector of the magnitudes of each column.
--------------------------------------------------------------------------
   Form:
   m = Mag( u )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   u            (:,:)  Vectors

   -------
   Outputs
   -------
   m            (:)   Corresponding magnitudes

--------------------------------------------------------------------------
