CwValid:

--------------------------------------------------------------------------
   Checks for valid HSA chordwiths
--------------------------------------------------------------------------
   Form:
   yesNo = CwValid( ra, dec, er, g, ue )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   ra                     Right ascension
   dec                    Declination
   er                     Earth radius
   g                      Nominal cant angle
   ue                     Nadir vector

   -------
   Outputs
   -------
   yesNo                  0 if the chordwidth is invalid

--------------------------------------------------------------------------
