DWInputs:

--------------------------------------------------------------------------
   Brings up a window for entering the desired data information
--------------------------------------------------------------------------
   Form:
   h = DWInputs( action )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   action               'Sensor Data' or 'Orbit Data'

   -------
   Outputs
   -------
   h                    Handle to the window created
                        

--------------------------------------------------------------------------

Children:

Common: Time/JD2Date
SAAD: GUI/SAADTool
