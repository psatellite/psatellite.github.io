ValidData:

--------------------------------------------------------------------------
 Computes predicted measurements, plots predictions with
 actual measurements
--------------------------------------------------------------------------
   Form:
   [k1,k2] = ValidData(hSAData,dAngles,cWidths,sAngles)
--------------------------------------------------------------------------

   ------
   Inputs - 
   ------
   hSAData  struct      Contains data and processing types for HSA measurements
   dAngles    {2}       Dihedral angles (rads)
   cWidths    {2}       Chordwidth angles (rads)
   sAngles   (n,1)      Sun angles (rads)

   -------
   Outputs
   -------
   k1        (m,1)      Indices to valid measurements for HSA (or SSA) 1
   k2        (p,1)      Indices to valid measurements for HSA 2

--------------------------------------------------------------------------
