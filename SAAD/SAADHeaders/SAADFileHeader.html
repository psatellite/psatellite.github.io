SAADFile:

--------------------------------------------------------------------------
   Data file functions used by SAAD GUI
--------------------------------------------------------------------------
   Form:
   h = SAADFile( action )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   action               'OpenAddDeleteWindow' or 'GetDirectory' or
                        'SetButtons' or 'MakeFileList'

   -------
   Outputs
   -------
   h                    Handle to the window that was created, or
                        cell array of full file names
                        1 if they do 

--------------------------------------------------------------------------
