AddBias:

--------------------------------------------------------------------------
   Adds biases to the non-zero elements of measurement vectors
--------------------------------------------------------------------------
   Form:  
   varargout = AddBias( varargin )
--------------------------------------------------------------------------

   ------
   Inputs - 
   ------
   Meas1          Measurement vector 1 - any size or units
    ...
   MeasN          Measurement vector n- any size or units
   biases  (n)    Bias values to be added. Units must match measurement
                  vectors

   -------
   Outputs
   -------
   adjMeas1       Adjusted measurement vector 1
    ...
   adjMeasN       Adjusted measurement vector n

--------------------------------------------------------------------------
