function BDRotate ( in  a)

Check the inputs.

Copyright (c) 1993-1997 Princeton Satellite Systems, Inc. All rights reserved.

Rotates the blocks in a block diagonal until only positive eigenvalues are in the upper s(n/2,n/2) block.

Form

[J,a] = BDRotate(a)

Inputs

a                  Input upper triangular matrix

Outputs

J                  Rotation matrix                 
a                  Rotated matrix

References: Maciejowski, J.M. (1989). Multivariable Feedback Design. Addison-Wesley, Reading, MA, 382-384.
