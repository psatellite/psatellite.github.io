function quat ( in  x)

Create an object of class quat.

You can input:

4-by-1 matrix - assumed to be a quaternion
3-by-1 matrix - assumed to be an array of angles
3-by-3 matrix - assumed to be a transformation matrix

Form

q = quat( x )

Inputs

x       (4,1) Quaternion
(3,3) Matrix
(3,1) Array of angles

Outputs

q      (1,1) Object of class  quaternion.
