FFEccZExt:

Path: FormationFlying/EccDynamics

% Compute extreme z-values and associated true anomalies for given relative motion.

   Since version 7.
--------------------------------------------------------------------------
   Form:
   [z,nu] = FFEccZExt( e, D, dH, epsilon, nMax );
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   e                (1)   Eccentricity
   D               (6,1)  Integration constants computed from i.c.'s
   dH               (1)   Integration constant found by setting H = 0 at nu0
   epsilon          (1)   Angular tolerance [rad]
   nMax             (1)   Maximum number of iterations

   -------
   Outputs
   -------
   z               (1,#)  Extreme values of cross-track amplitudes [km]
   nu              (1,#)  True anomaly where extreme amplitudes occur [rad]
