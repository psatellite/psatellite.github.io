% Initialize a delta-v command data structure.

   Since version 7.
--------------------------------------------------------------------------
   Form:
   cmd = DeltaVCommand_Structure( nThrusters );
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   nThrusters   (1)    Number of thrusters 

   -------
   Outputs
   -------
   cmd          (.)    Delta-v command data structure with fields:
                          - tank
                          - pulsewidth
                          - jD
                          - iD
