OrbitDataPlugin:

Path: FormationFlying/Visual

% Orbit Data Plugin for the Formation Design GUI.

   This function is called while initializing the Formation Design GUI.

--------------------------------------------------------------------------
   Form:
   d = OrbitDataPlugin( fig, pos, bColor, name );
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   fig              (1)   Handle to Formation Design GUI figure
   pos             (1,2)  Position data of GUI
   bColor          (1,3)  Background color of GUI

   -------
   Outputs
   -------
   d                (.)   Data structure of handles to "Orbit Data" panel controls

--------------------------------------------------------------------------
