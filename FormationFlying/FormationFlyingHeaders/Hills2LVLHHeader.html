Path: FormationFlying/Transformation

% Converts a state vector from the Hills to the LVLH coordinate frame.

   Since version 7.
--------------------------------------------------------------------------
   Form:
   xL = Hills2LVLH( xH )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   xH               (6,:) [r;v] Hills frame

   -------
   Outputs
   -------
   xL               (6,:) [r; v] LVLH frame
