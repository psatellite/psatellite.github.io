ViewEllipsoid:

Path: FormationFlying/Collision

% Plot collision monitoring ellipsoids.
 Plots ellipsoids for Collision Monitoring given the ellipsoid centers and
 matrices for 1 or 2 spacecraft.

--------------------------------------------------------------------------
   Form:
   ViewEllipsoid( xhat, Shat, dMin )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   
   xhat      {2}(6,n)      Ellipsoid centers, for both spacecraft
   Shat      {2}(6,6,n)    Ellipsoid size matrices, for both spacecraft
   dMin      {}(1,n)       Minimum distance between Ellipsoids predicted  

--------------------------------------------------------------------------
