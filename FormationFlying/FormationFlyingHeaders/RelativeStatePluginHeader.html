RelativeStatePlugin:

Path: FormationFlying/Visual

% Relative State Plugin for the Formation Design GUI.

   This function is called while initializing the Formation Design GUI.

--------------------------------------------------------------------------
   Form:
   d = RelativeStatePlugin( fig, pos, bColor, name );
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   fig              (1)   Handle to Formation Design GUI figure
   pos             (1,2)  Position data of GUI
   bColor          (1,3)  Background color of GUI
   name            (1,3)  m-file name of GUI for callbacks

   -------
   Outputs
   -------
   d                (.)   Data structure of handles to "Relative State" panel controls

--------------------------------------------------------------------------

Children:

Common: Graphics/NewFig
