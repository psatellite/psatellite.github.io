ManeuverStruct2AccelVector:

Path: FormationFlying/Utility

% Compute a 3xN acceleration vector from a "maneuver" data structure. 
   A constant timestep is used.

   Since version 7.
--------------------------------------------------------------------------
   Usage:
   [aC,t] = ManeuverStruct2AccelVector( mvr, dT );
   [aC]   = ManeuverStruct2AccelVector( mvr, t );
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   mvr           (.)      Maneuver data structure
   dT        (1) or (N)   Time-step or time vector                     [sec]

   -------
   Outputs
   -------
   aC           (3,N)     [x;y;z] acceleration over N time intervals   [km/sec^2]
   t            (1,N)     Time vector                                  [sec]
   
--------------------------------------------------------------------------
