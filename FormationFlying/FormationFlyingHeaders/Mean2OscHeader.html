Mean2Osc:

Path: FormationFlying/Transformation

% Transforms mean orbital elements to osculating orbital elements.
   Element vector in Alfriend form: [a,theta,i,q1,q2,W]

--------------------------------------------------------------------------
   Form:
   el_mean = Mean2Osc(el_osc);
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   el_mean          (1,6)    mean orbital elements

   -------
   Outputs
   -------
   el_osc           (1,6)    osculating orbital elements

--------------------------------------------------------------------------
   References:	Terry Alfriend, Texas A&M University, 2001
               Rich Burns, 1999
--------------------------------------------------------------------------
