Path: FormationFlying/DataStructures

% Initialize an eccentric geometry data structure.

   Since version 7.
--------------------------------------------------------------------------
   Form:
   g = EccGeometry_Structure( nU );
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   n            (1)    Number of geometry structures

   -------
   Outputs
   -------
   g            (.)    Geometry data structure with fields:
                          - y0
                          - xMax
                          - nu_xMax
                          - zMax
                          - nu_zMax
