FFEccHills2DeltaElem:

Path: FormationFlying/Transformation

% Compute element differences from Hills frame state and ref. elements

   Since version 7.
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   el0           (1,6)  reference orbital elements
   xH            (6,:)  relative position and velocity in Hills frame [km; km/s]
   type           (1)   type of orbital element set
                          1 -- [a,th,i,q1,q2,W]
                          2 -- [a,i,W,w,e,M]

   -------
   Outputs
   -------
   dEl           (:,6)  Orbital Element differences
                          if type 1 -- delta [a,th,i,q1,q2,W]
                          if type 2 -- delta [a,i,W,w,e,M]
   
--------------------------------------------------------------------------
