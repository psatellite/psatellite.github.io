ScaleState:

Path: FormationFlying/Coord

% Scale a relative state represented by a geometric goal set. This
   changes the size of the relative state, but not the phase.

   Since version 7.
--------------------------------------------------------------------------
   Form:
   g = ScaleState( g, s );
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   g            (.)    Geoemtric goals data structure (see Geometry_Structure)
   s            (1)    Desired scale factor. Existing state will be scaled
                       up (>1) or down (<1) by this amount. Only the
                       real absolute value of this number is used.

   -------
   Outputs
   -------
   g            (.)    New geoemtric goals data structure, scaled by "s".

--------------------------------------------------------------------------
