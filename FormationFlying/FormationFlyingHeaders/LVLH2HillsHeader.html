LVLH2Hills:

Path: FormationFlying/Transformation

% Converts a state vector from the LVLH to the Hills coordinate frame.

 This is a fixed transformation i.e. 
 Hill's x is LVLH -z
 Hill's y is LVLH  x
 Hill's z is LVLH -y

--------------------------------------------------------------------------
   Form:
   xH = LVLH2Hills( xL )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   xL               (6,:) [r;v] LVLH frame

   -------
   Outputs
   -------
   xH               (6,:) [r;v] Hills frame

--------------------------------------------------------------------------
