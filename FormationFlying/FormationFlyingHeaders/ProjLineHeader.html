ProjLine:

Path: FormationFlying/Coord

% Calculates orbital elements for an in-line formation.

--------------------------------------------------------------------------
   Form:
   elSet = ProjLine( aRef, iRef, kSats, rF ) 
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   aRef           (1,1)     Semi-major axis of reference orbit [km]
   iRef           (1,1)     Inclination of reference orbit [rad]
   kSats          (1,1)     Number of satellites
   rF             (1,1)     Distance between satellites [km]

   -------
   Outputs
   -------
   elSet         (kSats,6)     Orbital elements of all spacecraft
   rH            (3,kSats)     Radius vector in Hills frame   [km]
   vH            (3,kSats)     Velocity vector in Hills frame [km/s]

--------------------------------------------------------------------------
