NOrbVector:

Path: FormationFlying/Utility

% Compute a vector of maneuver durations from time window data

   Since version 7.
--------------------------------------------------------------------------
   Form:
   nOrb = NOrbVector( window );
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   window           (.)  Time window data structure with following fields
                          - nOrbMin
                          - nOrbMax
                          - nManeuvers

   -------
   Outputs
   -------
   nOrb
