RemoveStage:

--------------------------------------------------------------------------
   Remove a stage from a multi-stage data structure.
   Since version 1.0
--------------------------------------------------------------------------
   Form:
   d = RemoveStage( d, k )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   d             (.)   Initial multi-stage data structure
   k             (1)   Index of stage to be removed

   -------
   Outputs
   -------
   d             (.)   Updated data structure without stage k

--------------------------------------------------------------------------

Children:

Propulsion: MultiStage/MS2M
