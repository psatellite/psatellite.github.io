VonBraunRocket:

--------------------------------------------------------------------------
   The fictional Von Braun three-stage space shuttle. 

   This model is not in the database because it is so much larger than any
   physical rocket in existence, however it is used in examples in many
   textbooks. This design used nitric acid and hydrazine in all three
   stages.  The rocket would have stood over 80 m tall.

   Since version 2015.1
--------------------------------------------------------------------------
   Form:
   d = VonBraunRocket
--------------------------------------------------------------------------
   -------
   Outputs
   -------
   d              (.)        Rocket data structure

--------------------------------------------------------------------------
	Reference:	Wiesel, W. E., Spaceflight Dynamics, McGraw-Hill, 1988,
               pp. 196-197 and table 7-2.
