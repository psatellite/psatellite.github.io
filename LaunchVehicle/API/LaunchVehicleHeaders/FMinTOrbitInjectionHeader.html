FMinTOrbitInjection:

--------------------------------------------------------------------------
   Function to be minimized by fmins for MinTOrbitInjection.
   Since version 1.0
--------------------------------------------------------------------------
   Form:
   y = FMinTOrbitInjection( x, d )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   x           (2,1) [x;y;vX;vY]
   d            (.)  Data structure
                     .f1 (1,1) 2*altitude*height/final horizontal v^2
                     .f2 (1,1) acceleration/gravitational acceleration;

   -------
   Outputs
   -------
   y           (1,1) Cost

--------------------------------------------------------------------------
