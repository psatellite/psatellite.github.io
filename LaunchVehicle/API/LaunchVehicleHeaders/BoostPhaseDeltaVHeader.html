BoostPhaseDeltaV:

--------------------------------------------------------------------------
   Compute DV required to enter circular orbit at hF given V0 and gamma0.

   Assumes two delta-v's are applied. First at h0, then at hF.

   Since version 2015.1
--------------------------------------------------------------------------
   Form:
   [dV,dV1,dV2,gammaF] = BoostPhaseDeltaV( v0I, gamma0, h0, hF, rE, mu )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   v0I        (1,1)     Initial velocity, before the first delta-v (km/s)
   gamma0     (1,1)     Initial flight path angle (rad)
   h0         (1,1)     Initial altitude (km)
   hF         (1,1)     Final altitude for orbit (km)
   rE         (1,1)     Planet radius (default earth)
   mu         (1,1)     Planet gravitational constant  (default earth)

   -------
   Outputs
   -------
   dV         (1,1)     Total delta-v, dV1 + dV2 (km/s)
   dV1        (1,1)     First delta-v at h0 (km/s)
   dV2        (1,1)     Second delta-v at hF (km/s)
   gammaF     (1,1)     Flight path angle at hF (km/s)

--------------------------------------------------------------------------
   See also OrbitVel2D
--------------------------------------------------------------------------
