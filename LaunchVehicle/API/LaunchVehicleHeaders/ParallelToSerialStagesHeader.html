ParallelToSerialStages:

--------------------------------------------------------------------------
   Separate 2 parallel stages with partial overlap, into 2 serial stages.
   Assume p1 and p2 start at same time. 

   Since version 2015.1
--------------------------------------------------------------------------
   Form:
   [s1,s2] = ParallelToSerialStages(p1,p2,m0)
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   p1      (.)     Parallel stage 1
   p2      (.)     Parallel stage 2
   m0      (1,1)   Initial wet mass (same for both stages)

   -------
   Outputs
   -------
   s1      (.)     First serial stage
   s2      (.)     Second serial stage

--------------------------------------------------------------------------
