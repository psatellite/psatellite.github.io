SetPayloadMass:

--------------------------------------------------------------------------
   Set payload mass in structure d.

   Since version 2015.1
--------------------------------------------------------------------------
   Form:
   d = SetPayloadMass( d, mPLD )
--------------------------------------------------------------------------

   ------
   Inputs
   ------
   d              (.)        Rocket data structure with fields
   mPLD          (1,1)       New payload mass (kg)

   -------
   Outputs
   -------
   d              (.)        Updated data structure

--------------------------------------------------------------------------

Children:

LaunchVehicle: MultiStage/MSData
