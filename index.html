C/C++ Frameworks API

Our C/C++ Frameworks start with our MatrixLib matrix class library. Our ControlDeck framework builds lightweight multi-rate control systems. The DSim extensible framework provides simulation capability that integrates with ControlDeck. We provide a sample code library for custom control and simulation models. VisualCommander provides a graphical user interface for Mac OS X; the frameworks themselves are cross-platform.
